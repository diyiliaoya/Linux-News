XWayland 24.1.13 发布，修复 X.Org 代码库中的两个安全漏洞

XWayland 24.1.13 于 2026 年 7 月 7 日发布，修复了 X.Org Server 代码库中的两个安全漏洞：CVE-2026-55999（GLAMOR 字体代码缓冲区溢出）和 CVE-2026-56000（GLX 代码路径释放后使用）。这些漏洞由 Trend Micro Zero Day Initiative 借助 Claude Opus 4.6 发现，Red Hat 的 Peter Hutterer 宣布了修复。新版本还包含其他一些修复，X.Org Server 21.1.24 预计也将很快发布。
