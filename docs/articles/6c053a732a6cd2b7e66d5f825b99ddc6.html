Ubuntu 26.10 Snapshot 2 发布，用于月度测试

2026年6月25日晚上8点25分（美国东部时间）

Ubuntu 26.10 "Stonking Stingray" 的每日 ISO 仍在持续发布，但对于那些偏好更规范版本的用户，今天发布了 Ubuntu 26.10 Snapshot 2，作为第二个月度 ISO 镜像。

Canonical 继续使用 Ubuntu Linux 的月度 ISO 快照来帮助测试其发布基础设施。Stonking Snapshot 2 于今天发布，是 Ubuntu 26.10 当前开发状态的六月快照。

现阶段并不特别令人兴奋，因为在 10 月 Ubuntu 26.10 稳定版发布之前，未来几个月还会有大量更新。Ubuntu 26.10 计划搭载 GNOME 51、Linux 7.2 以及众多其他更新，超越 Ubuntu 26.04 LTS。但那些想了解 Ubuntu 26.10 Stonking Stingray 当前样子的用户，请查看 Snapshot 2 的下载详情。
