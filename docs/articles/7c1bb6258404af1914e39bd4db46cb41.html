m0n0wall 是一个旨在创建完整嵌入式防火墙软件包的项目，当与嵌入式 PC 配合使用时，它能够以极低的成本（自由软件）提供商业防火墙设备的所有重要功能（包括易用性）。m0n0wall 基于精简版 FreeBSD，并包含一个 Web 服务器 (thttpd)、PHP 以及其他一些实用工具。整个系统配置存储在一个单独的 XML 文本文件中，以保持透明。m0n0wall 可能是第一个在启动时使用 PHP（而非通常的 shell 脚本）进行配置，并且将整个系统配置存储在 XML 格式中的 UNIX 系统。
