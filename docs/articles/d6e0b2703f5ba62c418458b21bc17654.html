DXVK 3.0 发布：采用 DXBC-SPIRV 进行着色器编译，默认启用描述符堆

2026年6月25日，美国东部时间下午4:29。

Philip Rebohle 今天宣布发布 DXVK 3.0，这是基于 Vulkan API 的 Direct3D 8/9/10/11 实现的最新重大功能版本，用于 Wine 和 Valve 的 Steam Play（Proton）。

DXVK 3.0 的一大变化是现在使用 DXBC-SPIRV 进行着色器编译，以取代传统的着色器翻译代码。DXBC-SPIRV 是一个基于 SSA 的编译器，适用于 D3D SM 5.1+，同样由 DXVK 开发者开发。这个新的着色器编译器应能修复许多渲染问题，生成的代码比原生翻译的 SPIR-V 更紧凑，并且着色器编译完全卸载到工作线程，从而有助于缩短游戏加载时间。

DXVK 3.0 的另一大变化是，在支持的 Vulkan API 驱动上，现在默认使用 Vulkan 描述符堆（VK_EXT_descriptor_heap）。

DXVK 3.0 还带来了许多 Direct3D 改进、共享资源增强、帧率限制器支持，以及各种其他错误修复和改进。相应地，DXVK 3.0 现在需要 Vulkan 1.4 驱动级别支持。
