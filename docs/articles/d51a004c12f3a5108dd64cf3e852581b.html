Mozilla 发布了 Firefox 153，内置容器功能、二维码生成以及地址栏中更多快捷操作。

这是 Mozilla 转向更快发布节奏前的最后一个“月度”稳定版，之后将每两周发布一次新的 Firefox 稳定版。另外值得一提的是，在沉寂一年后，Mozilla 决定恢复在 X（原 Twitter）上的活动。

与所有更新一样，你可以为 Windows、macOS 和 Linux，以及 Android 和 iOS 获取此版本（移动版功能不同，本文不涉及）。

Firefox 内置容器功能

让你的浏览保持分离

容器是 Firefox 153 中最大的新功能。Mozilla 表示，这些容器让你“按容器分离 cookie，从而可以在同一网站上使用不同账户，并限制跨站追踪”。

例如，在同一窗口中同时使用个人和工作社交媒体账户，无需切换不同窗口或安装附加组件来避免账户间的交叉污染。

Mozilla 多年来一直提供容器功能。在 Firefox 153 中，核心容器功能已内置。默认设置了四个预配置容器，你也可以自己添加新容器。

创建你自己的容器，设置名称、颜色和图标，然后右键点击（或按住并点击）新标签页按钮，选择一个容器。或者，右键点击现有标签页，选择“在容器标签页中打开”，然后从中选择一个。容器标签页带有颜色编码，并在 URL 栏中显示图标。

如果你喜欢容器，可以前往设置页面进一步减少操作步骤。在那里，你可以找到一个选项，每次按下新标签页按钮时自动打开容器菜单。

二维码和 PDF 增强

在 Firefox 153 中为打开的标签页生成二维码

如果你想通过二维码分享网站（但觉得像 bar.codes 这样的在线二维码生成器太麻烦），Firefox 允许你右键点击标签页，选择“分享 > 生成二维码”来生成一个。不过，请注意，仅用于离线需求，因为在线分享链接仍然方便得多。

Mozilla 也在继续完善 Firefox 内置的 PDF 编辑功能集。今年五月，Firefox 增加了 PDF 合并功能，但依赖于文件选择器来选择要合并的 PDF。在 Firefox 153 中，你可以通过将 PDF 拖入侧边栏来合并。你还可以将图片作为新页面添加，这对于添加文档封面和图表非常有用。

Linux 上的圆角设计

四个角都匹配

眼尖的 Linux 用户可能会注意到，Firefox 在 GTK 桌面（如 Ubuntu）上默认启用了窗口底部圆角。多年来你可以手动启用此设置，但现在它默认开启——至少在我尝试的二进制构建中是这样。

上周发布，如果你有兴趣可以试试。目前尚未与桌面版功能完全一致（因为通知功能不起作用），因此被标记为实验性。

在打开的标签页上选取颜色，将值复制到剪贴板

Firefox 153 的其他变化包括地址栏中更多的快捷操作。如果你输入“labs”或“experiment”并点击“labs”操作，可以访问 about:labs 页面。要从正在查看的网页中复制颜色，请在地址栏中输入“pick colour”或“eyedropper”，然后使用“pick a colour”操作来获取取色器。

如果网站有权查看你的位置，浏览器现在会通过将相关图标高亮为红色来通知你，以便你知道该权限处于活动状态。这扩展到了搜索结果页面，据 Mozilla 称，此前该功能在此处是隐藏的。

在 macOS 上，Firefox 现在支持 Apple 的全系统全屏键盘快捷键（Control+Command+F），而 Windows 版 Firefox 153 现在支持 HDR 视频播放（在 Windows 本身支持并启用的情况下）。

然而，Mozilla 指出，HDR 目前无法在仅支持“HDR 视频流”的笔记本电脑显示屏上工作，并表示某些以特定方向在智能手机上拍摄的视频可能仍会以 SDR 格式渲染。
