Coreboot + AMD openSIL 在 MSI Ryzen 主板上现已支持 Windows 11

发布于 2026 年 7 月 3 日 上午 11:10 美国东部时间。

随着 3mdeb 将 AMD openSIL 和 Coreboot 的 Dasharo 移植版本部署到技嘉 EPYC 主板，3mdeb 工程师们投入了更多时间在面向桌面 AMD Ryzen 的 MSI PRO B850-P 消费级主板上推进 Coreboot+openSIL 的适配工作。如今，他们甚至已让微软 Windows 11 在这款开源固件替代方案上成功运行，同时还实现了其他功能。

除了将 Coreboot 和 AMD openSIL 原型引入该主板外，3mdeb 还独立地将固件栈移植到了 MSI PRO B850-P 上，这是一款现代 AMD Ryzen AM5 桌面平台。这使得 MSI PRO B850-P WIFI 成为 Linux 和开源爱好者的特殊/独特主板。

该固件移植近期取得的一些进展包括：解决了此前遇到的 AMDGPU 图形驱动错误、集成了 AMD PSP 固件 TPM 支持、EZ Debug LED 支持、SMBIOS 内存信息报告等。

虽然 Linux 是 3mdeb 的主要关注点，但他们也在测试微软 Windows 11 以解决各种问题。Windows 11 现在可以无需任何变通方案即可在此设置上成功运行。他们还实现了其他固件组件，以确保 Windows 电源管理行为的正确性。

看到 Windows 11 在 Coreboot+openSIL 设置上运行令人欣喜，并且这对于使该固件更适用于大众用户至关重要。
