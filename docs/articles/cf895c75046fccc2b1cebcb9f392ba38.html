Firefox 152 发布：全新设置的界面与奇特的标签页静音方式 - OMG! Ubuntu

Mozilla 发布了 Firefox 152，带来了重新设计且更快捷的网页内容分享方式——此外，还有一种奇特的方式来静音嘈杂的标签页。

该更新自今日（2026 年 6 月 15 日）起在 Windows、macOS 和 Linux 上可用，同时也适用于 Android 和 iOS（移动版本具有不同功能，本文不涉及）。

Firefox 152 的主要变化是重新设计的设置页面。我们早就知道这一变化，因为 Mozilla 已经预告了一年多。该公司表示，新外观提供了“简化的组织、更清晰的分组以及改进的导航，便于自定义”。

由于许多用户觉得在浏览器设置中翻找令人望而生畏且困惑（尤其是随着更多自定义选项的添加），这是一次受欢迎的刷新。好的设计既关乎清晰和逻辑，也关乎美观。

但请提前警告你的肌肉记忆，因为“常规”页面已被移除，其选项已移至相关区域，如隐私与安全、搜索等。如果你找不到某个设置，可以使用搜索栏来定位。

Firefox 152 中的设置页面在视觉上与即将推出的 Firefox Nova 重新设计相关联，后者也采用了大量圆角、带有细边框的卡片、渐变和空白区域。

Firefox 152 还有什么新功能？

分享链接稍微更容易了

Firefox 152 现在可以更轻松地在隐私浏览窗口中选择性禁用跟踪器拦截。如果网页的某个元素或部分因跟踪器被拦截而显示异常或无法加载（例如新闻故事中的 Instagram 嵌入内容），这一功能非常有用。

Linux 和 Windows 用户现在可以从标签页上下文菜单中复制链接（右键点击标签页，鼠标移至“复制链接”），而无需先聚焦该标签页。当选中多个标签页时，此功能也有效，因此你可以一次性复制所有选中的链接。

Firefox 还引入了一个“发送标签页”工具栏按钮，尽管默认未启用。要添加它，请前往“更多工具 > 自定义工具栏”，然后从按钮面板中将其拖到工具栏上。

想知道 Firefox 152 中那种“奇特”的标签页静音新方式是什么吗？浏览器允许你在地址栏中输入——是的，输入——“mute”、“shush”或“sssh”，然后会显示一个按钮，点击即可静音所有打开的 Firefox 窗口中正在播放音频的标签页。

我通常不会让一堆音频同时播放，而且伸手去按键盘上的静音或暂停媒体键总体上会更快。有选择总是好的——尤其是那些能让你扮演图书管理员角色的选择……

（顺便提一下），Firefox 152 为克罗地亚语、英语（英国）、格鲁吉亚语、波斯语、斯洛文尼亚语、塔吉克语、塔米尔语、藏语、土耳其语、威尔士语和科萨语的下载版本内置了拼写检查词典。

Firefox 152 中的其他变化

在某些网站（如 WordPress）上编辑文本时，新增了“粘贴为纯文本”选项

从 Firefox 拖到桌面的图片现在能可靠保存

在多显示器设置中，Firefox 会在 Firefox 所在的显示器上打开新窗口

使用键盘或鼠标以更小的增量进行缩放

如果源标签页已关闭，PDF 下载会在后台标签页中打开

从右到左文本的箭头键选择现在按正确方向移动

WebAuthn 相关来源请求以在多个域上使用通行密钥

此外，还有一系列常见的网页兼容性和开发者相关改进。它增加了对 Web 通知的操作按钮支持，支持了 CSS 属性 overscroll-behavior，并为 Pointer Lock API 添加了 unadjustedMovement 选项，以便网站可以获取“原始鼠标移动数据”。

另外，还有一批新的安全修复，以保护我们的集体网页浏览安全。

本次更新没有新增 AI 功能，但 Firefox 新的智能浏览模式仍在开发中。你可以通过翻转相关的 about:config 值（搜索“Smart Window”找到它）来作为测试版功能进行测试。

如何更新到新版本的 Firefox？

你可以从官方网站下载适用于 Windows、macOS 和 Linux 的 Firefox 152，提供 Intel/AMD 和 ARM64 设备的构建版本。

如果你已经在使用 Firefox，请以与上次更新相同的方式更新到最新版本——在大多数情况下是自动更新。

在 Windows 和 macOS 上，你也可以通过“帮助 > 关于 Firefox”手动检查更新。
