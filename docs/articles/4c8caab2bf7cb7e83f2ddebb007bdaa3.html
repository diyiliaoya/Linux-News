Linux 7.2-rc3 为 SEGA Dreamcast 驱动程序带来修复（2026年）

发布于 2026年7月12日 上午06:31 EDT

本周我并未预料到，但在今天 Linux 7.2-rc3 内核发布前合并到 Git 中的，是大量针对 SEGA Dreamcast 驱动程序的修复。

即使主线 Linux 内核开始放弃对许多老旧硬件平台的支持，在 SEGA Dreamcast 上运行 Linux 仍然可行——只需制作一张 CD-R 上的定制 Linux 构建，并使用 Dreamcast 键盘和鼠标进行交互。正是这些 Dreamcast 键盘和鼠标驱动程序在2026年迎来了意料之外的修复。

Linux 输入子系统维护者 Dmitry Torokhov 为 Dreamcast 的键盘、鼠标和摇杆驱动代码提交了多项修复，以正确处理驱动数据和设备注册，从而避免 Linux 内核中的一些竞态条件。Florian Fuchs 在修复了 SEGA Dreamcast “Maple”鼠标驱动代码中的内核崩溃后，推动了最近的 Dreamcast 更新。

这款1990年代末游戏机的 maplemouse 驱动代码中，因空指针解引用导致的内核崩溃问题自2017年以来一直存在。

今年早些时候，Linux 内核还修复了 Dreamcast 使用的 GD-ROM 驱动程序。同样在今年早些时候，有人提出了用于 Dreamcast 视觉存储单元（Visual Memory Unit）中少量特殊格式化闪存的 VMUFAT 文件系统驱动程序。但由于 Linux 文件系统要求的收紧，VMUFAT 驱动似乎不会被上游接受。
