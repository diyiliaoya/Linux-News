发行版发布：Home Assistant OS (HAOS) 18.0

Home Assistant OS（简称 HAOS）已升级至 18.0 版本。HAOS 是一个独立开发的、基于 Linux 的操作系统，针对运行 Home Assistant（一款开源家庭自动化工具）进行了优化。它适用于多种流行平台，包括 Raspberry Pi 和 ODROID，以及通用的 x86_64 和 AArch64 系统。

Home Assistant OS 18.0 是一个重大版本，为所有平台带来了显著更新。所有目标的 Linux 内核已从 6.12 更新至 6.18，Docker 已升级至 v29.5.3 并附带 containerd v2.2.4，底层 Buildroot 已更新至 2025.02.14。磁盘映像的闪存速度现在显著加快，默认交换空间大小现在有上限，以避免在内存极低或极高系统上出现问题，并且 Raspberry Pi 引导加载程序固件现在可以直接从 Home Assistant 更新。建议所有用户升级。Raspberry Pi 引导加载程序（EEPROM）固件现在可以直接从 Home Assistant OS 检查和更新。Home Assistant Core 2026.7.0 还将将该固件作为更新实体暴露。请注意，只有在系统从 SD 卡启动时，或者通常是在 Raspberry Pi 5 以及带有 CM5 的 Home Assistant Yellow 上，才能更新固件。
