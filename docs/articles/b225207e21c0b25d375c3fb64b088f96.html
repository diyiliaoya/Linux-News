Collabora Office 26.04 保持 AI 可选，并优化 Writer 和 Calc

Collabora Office 发布首个重大更新 26.04，基于 CODE 26.04 构建。新版本引入可选 AI 助手，用户可自选模型提供商或自托管，AI 不访问文档除非主动授权。Writer 和 Calc 获得多项改进：文档比较工具支持颜色编码，Calc 新增每用户工作表视图、表格设计选项卡和公式错误浮动助手。Impress 支持跟随式演示、幻灯片分组和多种尺寸混合。企业支持将在数周内推出。

Collabora Office 26.04 保持 AI 可选，并优化 Writer 和 Calc

桌面套件的首个重大更新从 CODE 26.04 中引入了多项功能。

Collabora Productivity 去年进入桌面编辑器市场，推出了基于与 Collabora Online 相同渲染机制但专注于离线使用的办公套件。他们这样做的目的是让用户无论在线还是离线都能获得相同的编辑体验，无需重新学习界面操作方式。其底层仍然是 LibreOffice 代码，但不再使用传统的 VCL 界面，而是采用了 JavaScript、CSS、WebGL 和 Canvas 技术栈。

这种共享代码库意味着 Collabora 产品之间的更新速度很快，这也正是 Collabora Office 的首个重大更新已经从 CODE 26.04 中引入功能的原因。

精炼的文档套件

插图来源：Collabora Productivity。

在没有加载文档的情况下打开三个编辑器中的任意一个，你将进入新的启动屏幕。它会显示带有文件类型图标的最近文档、模板库，并在文件打开后提供 AI 和文档签名等快速访问功能。

这里更大的新增功能是 AI。我知道你已经厌倦了看到其他开源办公套件纷纷染指 AI，但别担心，Collabora 保持了 AI 的可选性。

启用 AI 意味着你可以选择自己的模型提供商、自托管模型，或者选择你已信任的供应商，并输入自己的凭据。Collabora 完全不参与其中，除非你主动将文档交给助手，否则它无法访问你的文档。

启用后，AI 可以在 Writer 中起草和重写文本，在 Calc 中在你手动查找错误之前修复损坏的公式，并在 Impress 中将粗略的笔记转化为实际的幻灯片。它还可以生成图像，并在你没有时间手动浏览文档时进行摘要总结。

至于本次发布的其余部分，Office 26.04 包含了 CODE 26.04 版本带来的所有功能。在 Writer 中，你获得了一个重新设计的文档比较工具，它用颜色编码标记插入、删除和任何移动的内容，同时标注每次更改的修改者和时间。所有这些都可以通过并排视图或修订面板查看。此外，还有新的多页视图、更丰富的样式预览、导航器搜索以及 Markdown 导入/导出功能。

Writer、Calc 和 Impress 也都获得了大量更新。Calc 中的每用户工作表视图允许每个人设置自己的过滤器和布局，而不会影响他人；新的表格设计选项卡带来了合适的表格样式以及计算后的数据透视字段。公式错误现在会显示在单元格上的浮动帮助对话框中，因此你可以检查并修复问题，而无需滚动整个工作表。此外，还新增了一批函数，包括 XLOOKUP、XMATCH、LET 和 LAMBDA。在 Impress 中，跟随式演示允许观众自行向后滚动浏览之前的幻灯片，而不会跳到演示者前面。同样，幻灯片可以分组为章节，单个演示文稿可以混合多种幻灯片尺寸，多显示器支持也比以前更好。你还获得了更好的字体嵌入功能，确保演示文稿在任何地方打开时都能一致渲染。

Collabora Office 26.04 适用于多种平台，包括 Linux、Windows、macOS、Android 和 iOS。

要跟踪开发进度并访问源代码，你可以访问 Collabora 的 GitLab 仓库。

如果你正在寻找企业支持，Collabora 表示他们正在努力，并且本次发布是未来版本的预览。我问了 Collabora Productivity 的首席执行官 Michael Meeks，企业用户，特别是那些希望在 Linux 上部署 Collabora Office 的用户，可以期待什么。他回答说：

“企业现在就可以开始评估这个版本，支持将在几周内推出。他们可以期待更流畅的工作流程、更少的培训需求（得益于与 Collabora Online 共享的更美观且符合人体工学的用户体验）、内置的 AI 支持等等。我们期待企业用户的反馈。”
