GNOME Shell 当前在多显示器环境下只能使用同一张壁纸。开源开发者 Gianluca Masci 提交了实验性概念验证代码，通过架构改动实现每台显示器独立设置壁纸，并附带配置工具。该代码尚未合并到上游，但已托管在 GitHub 上，开发过程有 AI 辅助。

发布于2026年6月26日美国东部时间上午9:25。

GNOME当前多显示器处理的一个限制是，所有显示器使用相同的壁纸。对于那些希望享受每显示器独立壁纸选择的人来说，一些实验性/概念验证代码现在已能工作，允许在现代GNOME桌面上实现每显示器独立壁纸。

开源开发者兼Phoronix读者Gianluca Masci来信分享了他为GNOME Shell启用每显示器独立壁纸的概念验证工作。这为GNOME Shell内独立、每显示器的壁纸带来了必要的架构变更。

作为这项工作的一部分，还有一个用于GNOME Shell处理每显示器壁纸选择的配置工具。
