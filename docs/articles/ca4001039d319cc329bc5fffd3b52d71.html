Mesa 26.2-rc1 发布，本季度3D图形栈特性开发告一段落

发布于2026年7月15日美国东部时间下午5:10。

今天从 Mesa Git 分支出来，相应地，Mesa 26.3-devel 现在在主线上开放。

Mesa 26.2 正朝着8月的稳定版迈进，在此期间，每周的候选版本已经开始发布。Mesa 26.2 为 Intel ANV 和 AMD RADV Vulkan 驱动带来了多项优化，继续推进 Vulkan Video 工作，KosmicKrisp Vulkan-to-Apple Metal 驱动现已支持 Vulkan 1.4，Rusticl 持续提供更好的 OpenCL 支持，NVIDIA NVK Vulkan 驱动不断成熟，为 Arm Mali Panfrost/PanVK 驱动合并了新的 KRAID 编译器，开展了 AMD GFX12.1 图形支持的早期工作，并合并了多种新的 Vulkan 扩展。就连旧的 Radeon R600g 和 R300g 驱动在 Mesa 26.2 特性开发期间也获得了多项修复。

Mesa 发布说明总结了 Mesa 26.2 的亮点如下：

- radeonsi 上的 cl_khr_subgroup_rotate

- iris 上的 cl_khr_subgroup_rotate
