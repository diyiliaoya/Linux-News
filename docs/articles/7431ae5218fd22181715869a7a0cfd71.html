Linux 基金会等机构推出“Akrites”，以防御人工智能驱动的开源软件漏洞利用

Linux 基金会联合亚马逊、Anthropic、OpenAI、英伟达、微软、红帽等多家机构，共同发起 Akrites 项目。该项目旨在应对大语言模型快速发现新安全漏洞的挑战，通过建立共享安全事件响应团队和标准化协调漏洞披露流程，确保关键开源软件在漏洞被恶意利用前得到及时修复。初始支持者还包括谷歌、IBM、思科、摩根大通等。

Linux 基金会与亚马逊、Anthropic、OpenAI、英伟达、微软、红帽等机构联合发起 Akrites 项目。Akrites 项目旨在帮助防御关键开源软件免受人工智能/大语言模型快速发现的新软件漏洞和安全缺陷的威胁，确保这些问题在恶意行为者利用之前得到有效解决。

鉴于当前大语言模型发现新安全相关漏洞的速度惊人，Akrites 是一项行业范围内的努力，旨在确保关键开源软件能够及时得到缓解和加固。Akrites 的初始支持者包括亚马逊云服务、Anthropic、Chainguard、思科、花旗集团、Endor Labs、爱立信、谷歌、IBM、摩根大通、微软与 GitHub、英伟达、OpenAI、RapidFort、红帽、Rust 基金会、Sonatype、沃达丰和 Zscaler。

Akrites 正在建立：

“Akrites 建立一个共享的安全事件响应团队和一个单一的、标准化的协调漏洞披露流程，该流程基于保密优先原则和行业标准工具。

保密性是这项工作的核心。漏洞修复程序会按照维护者的意愿回流到每个项目的原始仓库。如果某个关键软件包没有活跃的维护者，Akrites 将作为最后的维护者，确保最新版本的修复程序及时送达所有人。该计划还将与政府工作协调，使公共和私人防御者协同行动。”

更多详情可通过今天的发布公告了解。
