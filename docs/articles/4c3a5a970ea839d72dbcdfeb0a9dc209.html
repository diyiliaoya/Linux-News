FEX 2607 针对尚未发布的 ARM 256 位 SVE2 硬件进行优化

发布于 2026 年 7 月 4 日 下午 03:40 EDT。

FEX 模拟器允许在 ARM64（AArch64）系统上运行 Linux x86/x86_64 软件，包括用于在 ARM 上进行 Windows 游戏运行的 Wine / Valve 的 Steam Play（Proton）。该模拟器现已推出最新的月度功能版本。这个由 Valve 支持的项目旨在为即将推出的 Steam Frame 及其他更典型的 ARM Linux 系统在 ARM 上运行 x86_64 游戏和其他软件，一直在酝酿更多优化和改进。

尽管目前尚无 256 位 SVE2 ARM 处理器可用，但 FEX 开发者一直在努力利用 256 位 SVE2 来实现其高级向量扩展（Advanced Vector Extensions），以便以高性能方式正确处理 AVX 指令。

FEX 2607 还为其 JIT 编译器带来了许多修复和改进。FEX 的另一个进步是开始支持 CUDA thunking，例如供在 DGX Spark 或其他 NVIDIA GB10 系统上使用 FEX 的用户使用。此 CUDA thunking 支持尚未完成，但可能开始适用于 ARM 上某些动态链接的 CUDA x86_64 软件。

FEX 还开始实现在 Proton/WINE 下的 Unixlib 支持，以克服当前将整个 FEX 构建为 Wine 加载的 DLL 文件所带来的一些挑战和工程限制。通过致力于 Unixlib SO 文件解决方案，未来将能够实现一些改进。
