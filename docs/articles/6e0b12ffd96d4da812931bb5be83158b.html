Linux Mint 现已将 Wayland 视为稳定版

Linux Mint 项目宣布，从下一个 Cinnamon 版本开始，Wayland 将不再被视为“实验性”功能，而是与 X11 一同获得完全支持。尽管 Mint 在 Wayland 采用上比 Ubuntu 和 Fedora 等发行版慢得多，但团队通过充分准备确保了稳定性。新版本将带来更清晰的图标、更好的鼠标光标、多显示器改进、硬件加速支持，以及重新设计的屏幕保护程序。此外，Linux Mint 23 已推迟至 2026 年圣诞节发布，以便为 Wayland 集成留出更多时间。

Linux Mint 现已将 Wayland 视为稳定版

Cinnamon 将在 Mint 的下一个版本（预计圣诞节发布）中完全支持 X11 和 Wayland 会话。

在主流 Linux 发行版中，Linux Mint 一直是个异类，多年来它一步一个脚印地谨慎推进，而 Ubuntu 和 Fedora 等其他发行版早已将 Wayland 设为默认体验。如今，这份耐心正在得到回报，该项目透露，从下一个 Cinnamon 版本开始，Wayland 将不再被视为“实验性”功能。

迟做总比不做好，对吧？

从下一个 Cinnamon 版本开始，X11 和 Wayland 会话都将得到完全支持，但后者不会成为默认会话。Linux Mint 的创始人 Clement Lefebvre 表示，Wayland 的体验现在感觉非常扎实，“几乎与 X11 持平”。

当然，达到这一步比其他发行版花了更长时间，但像 Mint 这样的项目不会在没有做好充分准备的情况下仓促进行如此重大的变更。你将在新版本中看到这些准备工作的成果：更清晰的图标、更好的鼠标光标，以及修复了影响 Slack 和 VS Code 等 Chromium 应用的 bug。

同样，窗口进度功能会在面板的应用按钮中显示 Nemo（文件管理器）的文件/文件夹复制进度，而防焦点窃取功能可防止其他应用在任务中途抢夺你的注意力。

多显示器设置和 KVM 切换器的表现更好了，硬件加速现在贯穿合成器、桌面会话以及 Wayland 和 Xwayland 客户端，包括针对 NVIDIA GPU 的 GBM over EGL。

今年二月，Lefebvre 透露团队正在重新考虑其发布计划，因为每六个月发布一个新版本，同时还要维护 LMDE，导致他们花在测试和发布上的时间比实际构建功能还要多。最终决定已定，Linux Mint 23 被推迟到 2026 年圣诞节，这是该项目主要版本之间最长的间隔。这部分额外时间直接投入到了 Wayland 上。

早些时候，Lefebvre 曾将重新设计的 Cinnamon 屏幕保护程序称为完全支持 Wayland 的最后一块拼图。旧的屏幕保护程序仅作为独立的 GTK 应用在 X11 上运行，位于 Cinnamon 自己的窗口管理器之外。新的屏幕保护程序在 X11 和 Wayland 上均可运行，并由 Cinnamon 自己的合成器原生渲染。

剩余的时间则为用户带来了更多便利。Mint 在五月开始提供 HWE ISO，让用户无需等待 LM 23 发布即可在 LM 22.3 上使用 6.17 等更新的内核。

现在只剩下等待了。基于已经完成的工作，下一个 Linux Mint 版本看起来值得期待。
