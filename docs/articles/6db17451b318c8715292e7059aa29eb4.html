KaOS 2026.06 发布

KaOS 项目宣布推出 KaOS 2026.06，标志着发行版全面转向 Dinit 初始化系统。经过近三年准备和一年测试，首个完全基于 Dinit 的稳定 ISO 镜像发布。KaOS 现使用 Dinit、turnstile、seatd 进行系统启动、席位和服务管理，但仍保留精简版 systemd 用于 udev 和 tmpfiles。显示管理器改用 greetd 与 tuigreet，取代 sddm。ISO 镜像默认搭载 Niri/Noctalia 桌面，而非 Plasma。用户仍可从仓库安装 Plasma 桌面。

发行版发布：KaOS 2026.06

KaOS 项目已宣布推出 KaOS 2026.06，该版本完成了发行版向 Dinit 初始化软件的迁移。

“经过近三年的准备以及大约一年的试验、构建、测试和重建，KaOS 自豪地向您呈现首个完全基于 Dinit 的稳定 ISO 镜像。对于那些对这些变更原因感兴趣的人，请阅读《systemd 与 KaOS 的未来》。脱离 systemd 作为初始化系统的迁移现已完成。KaOS 现在使用 Dinit、turnstile、seatd 进行系统启动、席位和服务管理。这并不意味着系统完全摆脱了 systemd，一个大幅精简的 systemd 仍主要用于 udev 和 tmpfiles（在可预见的未来将保持这种状态，其他用于 udev 和 tmpfiles 的选项将进行评估）。Elogind 仍在被使用，但并非用于系统登录，而纯粹是为了实现功能性的 polkit。一个新的显示管理器也已投入使用，greetd 与 tuigreet 已取代 sddm。Greetd 与 seatd 完全集成，因此在新设置中更适合 KaOS。您不会在此 ISO 镜像中找到 Plasma 或 Kwin，取而代之的是基于 Niri/Noctalia 的系统。”

希望在 KaOS 上继续运行 Plasma 的用户仍可从发行版仓库安装该桌面环境。更多信息请参阅 KaOS-DINIT-2026.06-x86_64.iso。

KaOS 是一款桌面 Linux 发行版，搭载最新版本的 Noctalia 桌面环境、Calligra 办公套件以及其他使用 Qt 工具包的流行软件应用。它受 Arch Linux 启发，但开发者构建自己的软件包，这些软件包可从内部仓库获取。KaOS 采用滚动发布开发模式，并专为 64 位计算机系统构建。
