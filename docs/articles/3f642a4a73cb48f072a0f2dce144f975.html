Redox OS 为 Orbital 桌面环境引入 GTK3 后端、分数缩放与 USB 游戏手柄支持

2026年7月8日下午2:30 EDT

开源、基于 Rust 的 Redox OS 平台在6月经历了一个非常繁忙的时期，实现了许多新功能，并将更多软件移植到这个从零开始构建的操作系统上。

Redox OS 今天发布了月度状态报告，介绍了这一创新开源操作系统的新变化和更新。6月的一些亮点包括：

- Redox 获得了 NGI Zero Commons 基金和 NLNet 的资助，用于开发虚拟化 Redox 支持。该工作重点在于让 Redox OS 能够在 Rust-VMM / QEMU 容器内作为 Web 服务器和微服务运行时运行。

- Redox 初步支持 USB 游戏手柄，具备通用和 Xbox 布局处理能力。此外，Redox 上更多 USB 设备现在也获得了更广泛的兼容性。

- 为 Redox 的 Orbital 桌面开发了 GTK3 后端，以提升性能、减少错误并消除 X11 开销。

- Orbital 现在也支持分数缩放，以便更好地适配 HiDPI 显示器。

- Tcl 编程语言现已移植到 Redox OS。

- Flycast 模拟器现在可以在 Redox OS 上运行，允许在该平台上运行各种游戏。

- 开始优化 Redox 的 EEVDF 调度器。

- 改进了路径处理性能。

- 其他各种系统增强以及对 Relibc 的改进。
