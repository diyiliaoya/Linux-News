Pine64 为 Home Assistant 爱好者推出 50 美元智能音箱 - OMG! Ubuntu

开源硬件制造商 Pine64 推出了一款售价 50 美元的智能音箱，该音箱在 RISC-V 芯片上运行开源软件。

（此前称为 PineVox）围绕 Bouffalo Lab BL606P RISC-V SoC 构建，集成了 Wi-Fi、蓝牙 5.0 和 Zigbee 无线接口。

它配备双麦克风阵列和扬声器，支持“本地唤醒词检测”，顶部按钮可实现静音（带 LED 指示灯）、启动/停止和音量调节。

出厂固件基于阿里巴巴的开源 YoC 平台，并运行 Wyoming Satellite 协议，该协议将设备转变为自托管、基于 Linux 的 Home Assistant 系统的本地麦克风和扬声器。

Home Assistant 可作为 Snap 包运行在 Ubuntu Core 上。

虽然 Pine64 表示这使用了 Wyoming 协议，但该协议已被弃用。开发者建议使用 Linux Voice Assistant。

仅配备 32 MiB 嵌入式 pSRAM 内存、16 MiB 闪存和 128 KiB ROM 存储，这些规格可能听起来微不足道——尽管在当前 AI 热潮下已算慷慨——但这是一款嵌入式设备，而非藏在香薰扩散器中的完整 PC。

PineVoice 并非全能型智能音箱。

PineVoice 并非竞争对手，它处于早期开发阶段，早期用户将遇到小问题和性能问题。未来的固件更新应能逐步解决这些问题，但像 Pine64 的所有产品一样，你购买的并非消费级产品。

时尚的工业设计并非 Pine64 的招牌，功能完善的软件也不是。该公司专注于价格：目标是提供廉价、开放友好的硬件，在开发者和爱好者手中，这些硬件可以发展成可行（尽管仍属小众）的产品。

烙铁也遵循同样的模式。

售价 50 美元，与 Amazon Echo Dot（第 5 代）和已停产的 Google Nest Mini 价格相当。与那些依赖云生态系统的专有消费设备不同，PineVoice 专为本地开源网络而构建。

感兴趣吗？你可以在 Pine64 商店以 49.99 美元的价格购买 PineVoice。它附带 USB 线缆、快速入门指南和 30 天保修——以及你试图让它做点什么时获得的周末乐趣。

也许是我个人看法，但它的设计有点……怎么说呢。放在某人浴室的架子上也不会显得突兀。对于一款智能音箱来说，低调的设计确实有道理。
