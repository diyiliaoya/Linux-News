发行版发布：OSINT VM 2026.5

Trace Labs 发布了 OSINT VM 2026.5，这是其基于 Debian 的发行版的最新版本，专为虚拟机设计，集成了开源情报（OSINT）工具，旨在帮助寻找失踪人员并与家人团聚。与之前基于 Kali Linux 的版本不同，此版本迁移至 Debian 13，移除了漏洞利用工具包、数据包分析器等非必要工具，专注于 OSINT 收集，从而减小镜像体积并提升控制力。
