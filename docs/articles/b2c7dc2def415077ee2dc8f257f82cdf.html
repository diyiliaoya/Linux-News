Ubuntu 26.04 修复 Papers 中 PDF 链接跳转错误页面的问题

Ubuntu 26.04 LTS 正在准备为文档查看器 Papers 发布一个错误修复更新，该更新解决了多个问题，包括内部 PDF 链接跳转到错误页面、快速滚动图像密集型 PDF 和 CBZ 时出现空白页、触控板捏合缩放触发意外滚动、分数缩放系统下的文字抖动以及铅笔图标大小异常等。Papers 50.2 更新将在未来几天作为标准软件更新推送。

Ubuntu 26.04 LTS 正在为文档查看器 Papers 准备一个错误修复更新，该查看器在 25.04 版本中取代了 Evince。此次更新将解决多个烦人问题——包括内部 PDF 链接跳转到错误页面的问题。

内部链接故障仅出现在部分 PDF 中，并非全部，并且通常会将你带到与实际目标相差一页的位置。这很烦人。修复方案阻止了两段独立的代码同时尝试在链接跳转到适应宽度或适应页面视图时设置缩放级别。

一个连带问题还导致 Alt+左箭头快捷键（用于跳回上一页）在点击有问题的链接后无法返回。

其他问题也得到了解决。

快速滚动图像密集型的 PDF 和 CBZ 文件时，经常会出现空白页飞速掠过的情况，直到你完全停止滚动——不过这只在启用分数缩放时发生。这里，渲染过程中使用的缩放计算是罪魁祸首。

继续与滚动相关的故障，在触控板上使用捏合缩放手势有时会触发意外的滚动——比如直接跳转到文档末尾。根本原因是 GTK 的一个错误，但已经找到了一个解决方法，即在缩放时取消惯性滚动。

分数缩放系统上的文字抖动以及铅笔图标（作为今年早些时候添加的注释工具的一部分）大小异常的问题也得到了修复。更低级别的怪癖和一个安全补丁也包含在内。

如果这些错误中的任何一个曾困扰过你，它们很快就会被解决。Papers 50.2 更新将在未来几天作为标准软件更新推送。无需任何特殊步骤即可获取；一旦准备好安装，软件更新器会通知你。
