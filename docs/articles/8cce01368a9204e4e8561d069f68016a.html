Guncrypt 是 Halfbrick Studio 的首款 PC 游戏，也支持 Linux

Halfbrick Studio 以《Jetpack Joyride》《Fruit Ninja》等手游闻名，其首款 PC 游戏《Guncrypt》是一款基于子弹排序机制的 rogue-like 地牢爬行者。游戏包含三种枪械、60 多种子弹和 80 多种遗物，支持组合与重排弹匣。塔罗牌和诅咒等级系统增加了策略深度。作者在 Nobara Linux 上通过 Steam Play 运行了演示版，无需额外文件即可流畅游玩。正式版尚无发售日期，但原生 Linux 版本正在开发中。

Guncrypt 是 Halfbrick Studio 的首款 PC 游戏，也支持 Linux

如果你喜欢 rogue-like 游戏，这款可能值得在 Linux 上入手。

Halfbrick Studio 最近联系了我们，他们兴奋地展示了最新项目《Guncrypt》，这是一款围绕按正确顺序装填子弹而非追逐更好装备的地牢爬行者。

如果这个名字听起来耳熟，他们正是《Jetpack Joyride》《Fruit Ninja》和《Dan the Man》等一系列热门手游的开发商。

在游戏中，武器系统包含三种枪械、超过 60 种子弹类型和 80 种遗物类型，你可以根据自身玩法风格组合它们来改变游戏体验。先装填普通子弹再装填火焰子弹，与先装填火焰子弹再装填普通子弹，效果截然不同。将两种子弹融合以获得新的组合效果，或者在房间之间重新排列整个弹匣，如果当前配置不奏效的话。

完成教程后的最初几分钟游戏画面。

塔罗牌在每次闯关之间为你的初始装备提供被动加成，而诅咒等级则在关卡不再具有威胁时，通过五个层级进一步增加难度。所有这些都发生在四个程序生成的地牢楼层中，每层都有独特的敌人、危险以及等待在尽头的 Boss。

在深入之前先了解一些背景信息。

我在我的 Nobara Linux 系统上运行了《Guncrypt》的演示版，游戏无需下载任何额外文件即可运行。这只是 Steam Play 利用 Proton 让游戏在这个非原生配置上运行。

无论如何，开场过场动画奠定了背景。Guncrypt 镇曾经是一个安静富裕的小镇，直到一个邪恶巫师出现，诅咒了它并偷走了所有人的灵魂，导致镇民们忙于互相争吵而无暇行动（听起来像时事）。结果任务就落到了玩家头上。

然后是教程，这对我来说是个意外，因为我习惯了 WASD 移动。这里完全没有这种操作；相反，我必须按住左键并拖动来移动我的枪手，同时射击他面前的任何东西。

进入游戏后，我与一些 NPC 互动，比如铁匠，然后进入了一个地穴，立刻看到了任务列表，其中一些任务可见。主要任务是揭开邪恶巫师施加诅咒背后的谜团。

开始一次闯关并被击败。

我进入了一个地牢，直接进入战斗，躲避敌人的同时留意房间之间的装填计时器。在一次让我吃尽苦头的闯关结束时，我看到了一个记分板，上面显示了我死亡时的楼层、使用的子弹以及各种评分标准的详细列表。

我解锁的一些东西。

子弹拾取机制类似。我一度遇到了三个选项，选择了闪电弹，这是一种 110 伤害的子弹，在装填时向敌人释放闪电。在游戏稍后部分，海盗“达斯蒂”给了我炸弹，这是一种有用的右键能力，用于炸毁敌人和障碍物。

安息吧达斯蒂，铁壳怪抓住了你。

截至撰写本文时，《Guncrypt》尚未公布发售日期或价格。Steam 页面仅将其列为“即将推出”。与此同时，你可以尝试演示版，它在我的测试系统上运行良好。

最后，原生 Linux 版本也正在开发中，将在正式版发布后不久推出。
