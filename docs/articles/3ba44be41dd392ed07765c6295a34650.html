发行版发布：Kali Linux 2026.2

Kali Linux 2026.2 正式发布，这是专注于安全与取证领域的 Linux 发行版的最新快照。本次更新带来了桌面环境升级至 GNOME 50 和 KDE Plasma 6.6、辅助脚本一致性改进、APT 格式迁移至 sources.list.d、虚拟机启动优化、重启警告、内核保持 6.19 版本并支持提前体验 7.0、构建脚本变更以及新增多个工具包。

发行版发布：Kali Linux 2026.2

Linux 项目已发布其专注于安全与取证领域的发行版的新快照。该项目的 2026.2 版本包含多项升级：

“这是第二季度的最后一周，Kali Linux 2026.2 如期而至。自上次发布以来，我们一直埋头工作，现在准备分享我们的成果。本次发布融合了桌面刷新、基础设施改进以及生活质量方面的变化，相信大家会喜欢。自 2026.1 版本（3 月发布）以来的变更日志摘要如下：桌面环境——升级至 GNOME 50 和 KDE Plasma 6.6；辅助脚本一致性——对启动服务的少量启动项进行一致性调整；APT 格式——告别 sources.list，改用 sources.list.d/kali.source；虚拟机启动优化——更小的 initrd 加上更快的启动时间，让虚拟机用户更满意；重启警告——请注意，polkit 和 xrdp 的升级需要重启系统；Kali 内核即将到来——目前仍使用 6.19 版本，并介绍如何提前获取 7.0 版本；构建脚本即将更新——提前告知即将到来的变更；新工具——一如既往，新增了多种闪亮的新软件包（共 9 个）。”
