厌倦了复杂的笔记应用？试试 Linux 上的 Whisp

Whisp 是一款专为 Linux 设计的 GTK4/libadwaita 笔记应用，定位为“反笔记”工具，强调极简和手势操作。它无需标题、文件夹或标签，笔记以可滑动的轮播形式呈现，支持 Markdown 和智能文本扩展。开发者称其为“为绝对速度设计的流畅手势便签本”，旨在摆脱数据库、层级等复杂组织方式。上线几周便在 Flathub 获得超过 2500 次下载，适合追求快速记录的用户。

厌倦了复杂的笔记应用？试试 Linux 上的 Whisp - OMG! Ubuntu

这款新的 GTK4/libadwaita 应用正将自己定位为那些厌倦了笔记应用（最好的永远是下一个，对吧？）的人们的笔记工具。

它自称是“GNOME 的反笔记”，借鉴了 macOS 上一款外观和功能集相似的应用。开发者将其描述为“一款为绝对速度设计的流畅、手势驱动的便签本”。

该应用的网站对复杂笔记应用进行了抨击，但 Whisp 并非要与它们竞争。它是对依赖数据库、层级结构和软木板加红绳式复杂组织的笔记方式的一种反制。

至于我？我是一个毫无章法的野蛮人。用笔和便利贴直到新鲜感消退，然后改用纯文本文件、手机笔记，偶尔还会在博客上敲出一篇草稿。

你不需要给笔记起标题——这个想法让我有点不安——也不需要将笔记保存为单独的文件。没有文件夹，也没有标签系统。

“只需打开应用，立即开始打字”

浏览笔记完全由手势驱动：

Whisp 的手势流程演示

没错，没有侧边栏；相反，你的笔记存在于一个可滑动的轮播中。向左或向右滑动，点击窗口边缘，或使用键盘快捷键在笔记之间切换。当你滑过最后一篇笔记时，会出现一篇空白笔记。

简洁的智能扩展器用于数据输入

Markdown 支持让你使用常见语法格式化笔记，并且有一个所见即所得的切换开关用于实时预览。

创建清单只需输入相关语法，尽管我找不到勾选项目的方法。

日期和时间的智能文本扩展（在最近更新中添加）让你输入特定内容以触发自动补全下拉菜单：

全局搜索可匹配所有笔记。通过标题栏中的按钮或按快捷键激活。点击搜索结果可直接跳转到该笔记。

其他功能包括：

- 链接折叠（折叠长链接）

- 可调节行间距

- 网格、横线或纯色背景

- 启动偏好设置（新笔记或恢复会话）

- 自动清理不活跃笔记（1个月、1年或从不）

- 全局快捷键启动（应用可在后台运行）

Whisp 是一个便签本，更像是一个临时记录工具，而非存档；它是午夜思绪倾泻、漫无边际的吐槽以及其他任何说出来比憋在心里更好的东西的容器。而且，由于几乎没有什么需要整理，你也不会因为整理而抓狂。

显然，Linux 用户对一款专注、现代的笔记工具有着强烈需求，因为 Whisp 在 Flathub 上线后的最初几周内就被下载了超过 2500 次。
