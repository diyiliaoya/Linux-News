OpenRazer 3.12.4 修复与 Linux 7.2 的兼容性

OpenRazer 3.12.4 发布，主要修复了与 Linux 7.2 内核的兼容性问题。由于 Linux 7.2 移除了 strncpy API，OpenRazer 将代码中的 strncpy 替换为 strscpy()，以确保正常运行。该版本无新增功能，但为 Razer 外设在 Linux 上的使用提供了持续支持。

OpenRazer 3.12.4 修复与 Linux 7.2 的兼容性

发布于 2026 年 7 月 4 日 晚上 8:33（美国东部时间）

OpenRazer 3.12.4 现已发布，这是针对 Razer 设备的树外非官方 Linux 驱动程序的最新更新。OpenRazer 与 Polychromatic 或其他 GUI 选项配合使用时，能够在 Linux 上为 Razer 游戏外设提供出色的使用体验。

OpenRazer 3.12.4 没有引入任何炫酷的新功能或新增 Razer 产品支持，其主要变化是启用了对最新 Linux 7.2 Git 内核状态的支持。

Linux 7.2 在经过多年工作和数百个补丁后，最终移除了 strncpy API，OpenRazer 也必须进行更新，因为其部分代码依赖于 strncpy。OpenRazer 3.12.4 版本现已使用 strscpy() 替代 strncpy()，以实现与 Linux 7.2+ 的兼容。
