Collabora Online 迎来十周年，拒绝将你的数据交给别人的AI

Collabora Online 发布 26.04 版本，庆祝其开源办公套件问世十周年。新版本引入 AI 辅助功能，覆盖 Writer、Calc 和 Impress，但强调用户可自主选择是否启用，并支持自托管或连接信任的商业 API，不强制绑定特定供应商。此外，协作功能增强，包括每用户工作表视图、数据透视表计算值、演示文稿同步视图，以及大幅改进的文档比较工具。Collabora 提供免费社区版、桌面版和企业版，企业版附带付费 SLA 和安全更新。

其最新版本吸收了 CODE 26.04 的改进，因此涉及了 AI。

Collabora Productivity

过去十年一直在证明，开源办公套件可以直接与 Microsoft 365 和 Google Workspace 竞争，而无需组织放弃对其基础设施或文档的控制。

这一切都得益于一支开源工程师团队，他们的工作使 Collabora 成为企业寻找协作文档平台时的可信赖品牌。

为庆祝十周年，

发布了 Collabora Online 26.04

，标志着这款可自托管产品的一个重要里程碑。

Collabora Online 26.04 包含 Writer、Calc 和 Impress。

如果你关注过今年六月发布的 CODE 26.04，那么 Collabora Online 26.04 中的大部分内容都会让你感到熟悉。所以这里我将重复我们之前已经介绍过的信息。

AI 辅助功能这次覆盖了套件中的每个主要应用，

落地于 Writer、Calc 和 Impress。Writer 的版本帮助起草和改写段落，Calc 的版本处理公式错误和电子表格数据，Impress 的版本将笔记或较长的文档压缩成适合幻灯片的材料。

除非组织先开启它，否则这些功能都不会运行。从那里开始，团队可以选择自己的路径，连接他们已信任的商业 API，或者在他们控制的基础设施上部署自托管模型，Collabora 不推销任何特定供应商。

CODE 26.04 的变更倾向于协作。每用户工作表视图允许两个人过滤和重新排列同一共享电子表格，而不会互相干扰设置。数据透视表现在支持计算值，并且为从事严肃数据处理工作的人提供了一批新函数。

演示文稿方面，观众可以自动跟踪演示者当前所在的幻灯片，并可以选择暂停和回看之前错过的内容，然后再重新加入实时视图。

Writer 的文档比较工具是本次周期中改动最大的部分。调出文件的早期版本，无论是位于服务器上还是本地，编辑器都会高亮显示与当前草稿相比的所有更改，并注明修改者和修改时间。

你还可以选择并排审阅更改，或通过通常的侧边栏审阅，并且还获得了 BITV 2.0 认证。

谈到此次发布，

Collabora Productivity 的 CEO 表示：

“Collabora Online 26.04 高度可定制且严格控制的 AI 辅助功能，使用户能够安全地加速完成与文档相关的任务。”

“同样，新的编辑和审阅功能、改进的可访问性以及更强的文档互操作性，将平滑所有用户的日常体验，使其更加直观和灵活。”
