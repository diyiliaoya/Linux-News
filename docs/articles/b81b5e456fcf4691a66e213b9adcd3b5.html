BSD 发布：FreeBSD 15.1

FreeBSD 15.1 正式发布，这是 stable/15 分支的第二个版本。主要亮点包括：基于 Linux 7.0 的无线网络驱动更新、云镜像支持首次启动时自动更新基础系统包、新增内核调度器启动时选择功能、C23 语言标准支持取得重大进展、Unicode 更新至 17.0.0 并新增 4803 个字符。此版本特别纪念长期合作者 Peter G. Neumann。

Colin Percival 宣布发布 FreeBSD 15.1，这是 FreeBSD 最新稳定分支的第二个版本：

FreeBSD 发布工程团队很高兴地宣布 FreeBSD 15.1-RELEASE 现已可用。这是 stable/15 分支的第二个版本。主要亮点包括：iwlwifi(4) 及其他基于 LinuxKPI 的无线网络驱动现已基于 Linux 7.0；使用打包基础系统的 FreeBSD 云镜像现在包含 pkg(8)，并支持首次启动时自动更新基础系统包；新增 kern.sched.name 可调参数，允许在启动时选择内核调度器；对 C 语言 C23 版本的完整支持取得了重大进展；Unicode 支持已更新至 Unicode 17.0.0 和 CLDR 48，新增 4803 个字符。FreeBSD 项目将 FreeBSD 15.1-RELEASE 献给 Peter G. Neumann 的纪念，他是 FreeBSD 开发者在基于能力的安全方面的长期合作者……

FreeBSD 是一个类 UNIX 操作系统，支持 i386、amd64、IA-64、arm、MIPS、powerpc、ppc64、PC-98 和 UltraSPARC 平台，基于加州大学伯克利分校的“4.4BSD-Lite”版本，并包含一些“4.4BSD-Lite2”增强功能。它还间接基于 William Jolitz 将加州大学伯克利分校的“Net/2”移植到 i386 的成果，即“386BSD”，尽管现在仅剩极少量 386BSD 代码。FreeBSD 被全球的企业、互联网服务提供商、研究人员、计算机专业人士、学生和家庭用户用于工作、教育和娱乐。FreeBSD 附带超过 20000 个软件包（预编译软件）。
