Academy Software Foundation 宣布成立“Wayland for Artists 工作组”

2026年6月25日下午2点12分（美国东部时间）

致力于推动视觉特效/电影行业及其他领域开源项目（如 OpenVDB、OpenMoonRay、Open Shading Language 等）的 Academy Software Foundation，宣布成立一个新的工作组，以帮助艺术家更好地采用 Wayland。

Academy Software Foundation 与视觉特效协会技术委员会共同组建了“Wayland for Artists 工作组”，就 Linux 上 Wayland 相关事宜展开合作。该工作组将成为一个行业范围的论坛，围绕视觉特效和动画行业进行变革，确保 Wayland 能够良好地服务于专业视觉特效和动画艺术家。

随着越来越多的 Linux 桌面工作从 X11 迁移到 Wayland，Wayland for Artists 工作组正致力于确保艺术家能够平滑过渡，且功能和特性上不出现任何缺口。

“Wayland for Artists 工作组将探索可能受到影响的多个行业特定领域，包括图形平板支持、窗口管理、输入焦点、键绑定、桌面定制、远程访问解决方案、GPU 加速、色彩管理、高动态范围显示，以及与商业、开源和内部应用的兼容性。”
