Debian 13.6 发布：集成所有最新安全修复，回退 GeoIP 数据库

2026年7月11日，美国东部时间上午10:31。

Debian 13.6 于今日发布，作为 Debian Trixie 的最新小版本，集成了最新的安全修复和其他维护更新。

Debian 13.6 集成了最新一批安全修复，包括涉及 Linux 内核以及 Nginx、Redis、FFmpeg、Thunderbird、Chromium、PHP 等多个软件包的安全公告。Debian 13.6 还包含了多项错误修复和其他更新。此外，Debian 决定将用于地理定位 IP 支持的 geoip-database 包回退至 2019 年 12 月的状态。来自 GeoLite 的较新版本 geoip-database 已被认定与 Debian 自由软件指南不兼容。

Debian 13.6 还引入了 fwupd 2.0.20，该版本增加了更新 Secure Boot CA、KEK 和 DBX 数据库的能力，原因是目前大多数 PC 默认使用的 UEFI Secure Boot CA 已经过期。
