新的 AMD Linux 补丁暴露 Gamma 2.4 和 Gamma 2.6 曲线

发布于 2026 年 6 月 24 日上午 10:22（美国东部时间）。

AMD 工程师终于忙于为其开源 Linux 驱动推出 HDMI 2.1 支持，与此同时，另一个显著的显示相关改进正在进入他们的 AMDGPU 内核图形驱动——暴露 Gamma 2.4 和 Gamma 2.6 曲线支持。

源于上个月在法国举行的 Display Next Hackfest 2026，来自未指明方对在 Linux 下看到 AMD 硬件暴露其 gamma 2.4 和 2.6 曲线表示了兴趣。昨天发送的补丁使这一目标在 AMDGPU Linux 驱动中成为现实。

AMDGPU 驱动目前暴露了标准的 gamma 2.2 支持，这是标准的 gamma 曲线。Gamma 2.4 在电影和电视行业中很流行，用于在客厅环境中提供更高的对比度和观看体验。Gamma 2.6 是数字影院和电影制作的官方标准，以其更深的阴影和饱和度实现更好的影院再现。

通过昨天发送的补丁，AMDGPU Linux 驱动暴露了 Gamma 2.4 和 Gamma 2.6，以及它们的逆曲线。这是一个直接的添加，为现代 AMD 图形处理器添加了 Gamma 2.4 和 Gamma 2.6 支持，并有望在今年晚些时候被纳入 Linux v7.3 内核。
