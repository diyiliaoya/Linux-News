Ubuntu 26.04 LTS vs. Windows 11 vs. CachyOS 在5399美元笔记本上的性能对比

发布于2026年7月15日美国东部时间下午12:15。

本月早些时候，我在 Phoronix 上评测了 Razer Blade 18 RZ09-0582，这是 Razer 首款通过 Canonical 的 Linux 硬件认证计划认证为 Linux 使用的笔记本电脑。它搭载 Intel Core Ultra 9 290HX Plus 和 NVIDIA GeForce RTX 5090 显卡，性能出色，但配置价格高达5399美元。该评测基于 Ubuntu 26.04 LTS 进行了基准测试，但如果您想知道在这款游戏/AI 开发笔记本上 Ubuntu 与 Windows 11 的性能差异，这里提供了对比基准测试，同时还加入了滚动发行版 CachyOS。

本文包含了 Microsoft Windows 11（Razer 出厂配置）与标准 Ubuntu 26.04 LTS 环境（搭载 Linux 7.0 内核及其他默认软件组件）的基准测试对比。最后，还在同一台笔记本上测试了注重性能的基于 Arch Linux 的 CachyOS 发行版，以了解这款笔记本上更前沿的 Linux 性能表现。

考虑到使用的 NVIDIA GeForce RTX 5090 显卡，所有测试均使用了各发行版仓库中提供的官方 NVIDIA Windows/Linux 驱动。未进行 Nouveau 对比，但针对 GeForce RTX 5090 笔记本 GPU 性能的独立 NVIDIA vs. Nouveau 对比即将推出。

截至上周，滚动更新的 CachyOS 使用了 Linux 7.1 内核、GCC 16.1.1，并已升级至 NVIDIA R610 驱动，而 Windows 11 和 Ubuntu 26.04 仍使用 R595 驱动。
