发行版发布：Univention Corporate Server 5.2-6

Univention 团队发布了基于 Debian 的企业级发行版 Univention Corporate Server 5.2-6。此版本整合了过去三个月的所有新功能和改进，包括来自 Univention Nubus 的新指标端点以提升可观测性，并在管理界面中新增了目录对象的技术标识符显示，方便管理员将技术 ID 与用户、组等对象匹配，从而简化配置问题排查。

发行版发布：Univention Corporate Server 5.2-6

团队发布了该项目基于 Debian 的 Univention Corporate Server 的新版本。最新版本改进了群组邮箱的处理方式，并使得在管理界面中识别对象更加容易。

“Univention Corporate Server 的最新补丁级别版本将过去三个月的所有新功能和改进整合到了新的安装介质中。UCS 5.2-6 带来了来自 Univention Nubus 的亮点功能，例如用于提升可观测性的新指标端点，以及在管理界面中显示目录对象的技术细节，并为邮件栈增加了群组的备用邮件地址。在分析配置问题或最终用户事件时，通常需要访问后端使用的技术信息，例如 Univention 对象标识符。为了简化将真实名称与用户、组以及目录服务中存储的任何其他信息的技术标识符进行匹配的过程，管理界面现在新增了一个包含这些标识符的板块。这在多种场景下都很有帮助：如果后端服务中记录了包含技术标识符的警告或错误，管理员现在可以直接在管理界面中搜索该标识符，并轻松访问受影响对象的完整信息。”
