HW Lab 的 DockFrame 携可更换工具卡登陆 Crowd Supply

HW Lab 推出模块化 USB-C 集线器 DockFrame，采用 Framework 扩展标准，支持可更换工具卡，兼具工作台工具功能。该设备提供四个开放插槽，兼容 Framework 的扩展卡，并自带电压电流测量、USB 3.0 扩展、可编程电源和原型开发等工具卡。产品注重可修复性，使用螺丝而非胶水固定，目标用户为创客、嵌入式工程师和教育工作者。众筹即将在 Crowd Supply 上线，硬件文件将在众筹结束后开源。

众筹是将创意转化为实际产品的最佳方式之一。创作者提出方案，设定筹资目标，由支持者决定是否值得打造。

虽然大多数众筹项目规模较小，但少数项目会彻底爆发。最初的 Pebble 智能手表在 2012 年寻求 10 万美元资金，最终筹得超过 1000 万美元，并成功出货超过 200 万只手表。

HW Lab 已经熟悉这条道路。其首款产品——面向游戏玩家和创作者的、可破解的副屏——在 Crowd Supply 上寻求 5250 美元，最终获得 10718 美元，超出目标一倍多。

现在他们带着一款专注于模块化、可修复的 USB-C 扩展坞的新产品再次回归。

仅仅是一个通用的 USB-C 集线器？

DockFrame 是一款围绕 Framework 插槽标准构建的模块化 USB-C 集线器。它没有固定的端口组合，而是提供四个开放插槽，可容纳 Framework 为其笔记本电脑和台式机销售的相同扩展卡。

该集线器本身通过 USB-C DisplayPort 和 Power Delivery 直通运行，下游端口最高支持 100W 供电。外壳采用注塑成型且为半透明材质，支持 Framework 的 LEGO 兼容螺柱网格底板，因此当四个插槽不够用时，可以将多个单元堆叠起来。

插入任何 Framework 扩展卡，DockFrame 都会像对待原生设备一样使用它们。这意味着笔记本电脑升级后剩下的任何备用 USB-A、HDMI、以太网或存储卡都不必闲置在抽屉里；它们可以直接插入集线器使用。

HW Media Lab 也自行构建工具卡。电压/电流/连续性卡可读取直流/交流电压、电流和通断。USB 3.0 扩展卡增加四个 USB 3.0 端口，每个端口支持 5 Gbps 传输速度。可编程电源卡是一款可编程升降压电源，支持最高 100W 的 USB PD 输入。原型开发卡搭载 Seeed Studio XIAO 板和标准 0.1 英寸排针，用于原型开发工作。

如果这四张卡都不符合你的需求，你可以使用相同的开放插槽标准自行设计扩展卡，或者构建一个自定义主机应用，通过 I2C 与板载 MCU 通信。

可修复性也是这里的重点，因为 DockFrame 使用螺丝而非胶水固定。创客、嵌入式工程师、教育工作者，以及任何厌倦了将扩展坞、万用表和台式电源作为三个独立盒子的人，都是其目标受众。

DockFrame 尚未公布正式发布日期或价格。在等待期间，你可以关注其 Crowd Supply 页面，如果想分享反馈，官方产品页面上有一个可填写的表单。
