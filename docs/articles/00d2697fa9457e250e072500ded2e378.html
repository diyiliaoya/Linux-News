高通为HP EliteBook X G2q X2 Elite笔记本电脑发布Linux补丁

高通工程师近日发布了针对HP EliteBook X G2q笔记本电脑的Linux支持补丁，该机型搭载Snapdragon X2 Elite SoC。补丁实现了Adreno图形、HDMI输出、USB Type-C、eDP、NVMe SSD、WiFi、键盘和触摸板等大部分功能，但集成摄像头尚未支持。该支持依赖于尚未合并的Linux内核补丁，有望在Linux v7.3内核中完成。

高通为HP EliteBook X G2q X2 Elite笔记本电脑发布Linux补丁

发布于2026年6月21日上午10:38 EDT。

上个月，高通工程师发布了在Linux上支持联想Yoga Slim 7x Gen11 Snapdragon X2笔记本电脑的补丁。本周末，高通又发布了一组新补丁，用于支持搭载Snapdragon X2 Elite SoC的HP EliteBook X G2q笔记本电脑型号。

HP EliteBook X G2q有Snapdragon X2 Plus和X2 Elite两种型号，后者现在通过高通工程师Jason Pettit提交的待定补丁获得了Linux支持。

他们在实际硬件上实现了Linux支持，并使大部分功能正常工作，包括Adreno图形、HDMI输出、USB Type-C、内部eDP、NVMe SSD、WiFi、键盘和触摸板以及其他功能。集成摄像头是少数尚未支持的功能之一，这与近期其他Snapdragon X2 Linux笔记本电脑的启用情况类似。

HP EliteBook X G2q配备14英寸WUXGA显示屏，基础型号标配512GB存储和24GB内存，但最高可支持64GB内存，并预装Microsoft Windows 11 Pro。在当前环境下，HP EliteBook X G2q基础型号售价为3,885美元，且该型号为X2 Plus版本，不受这些补丁支持。X2 Elite版本搭载高通Snapdragon X2E-84-100和32GB内存，售价4,586美元；而搭载X2E-90-100和64GB内存的旗舰型号售价则高达6,337美元以上。

此HP EliteBook X G2q的支持依赖于其他尚未合并的Linux内核补丁。我们将看到这一切能否在今年晚些时候的Linux v7.3内核中完成。无论如何，Snapdragon X2 Linux笔记本电脑的进展比Apple Silicon方面更快——后者Linux终于能在Apple M3上启动，但由于缺乏GPU支持和其他功能，目前尚无法供任何实际终端用户使用。
