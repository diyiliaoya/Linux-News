Razer 认证其首款 Linux 笔记本电脑：Razer Blade 18 RZ09-0582

Razer 正在认证其首款支持 Ubuntu Linux 的笔记本电脑——Razer Blade 18 RZ09-0582，搭载 Core Ultra 9 290HX Plus 处理器和 GeForce RTX 5090 显卡，性能强劲但价格高昂。该认证标志着 Razer 在 2026 年终于兑现了近十年前 CEO 对 Linux 支持的承诺。目前该设备在 Ubuntu 26.04 LTS 上运行流畅，但 Razer 尚未宣布将 Synapse 软件引入 Linux，且社区驱动的 OpenRazer 和 Polychromatic 暂不支持此型号。

Razer 认证其首款 Linux 笔记本电脑：Razer Blade 18 RZ09-0582

发布于 2026 年 7 月 7 日，美国东部时间上午 11:38。

Razer 是一个以游戏闻名的品牌，终于在 2026 年，他们正在认证首款支持 Ubuntu Linux 的笔记本电脑。这款正在通过 Ubuntu Linux 认证的笔记本电脑是 Razer Blade 18 RZ09-0582，它搭载了 Core Ultra 9 290HX Plus 处理器和 GeForce RTX 5090 显卡，性能惊人，但价格也非常高昂。

Razer 更好地拥抱 Linux 已是期待已久。早在 2017 年 3 月，Razer 首席执行官 Min-Liang Tan 就公开表达了在 Razer Blade 笔记本电脑上更好支持 Linux 的兴趣。他甚至希望将 Razer Blade 打造成“世界上支持 Linux 的最佳笔记本电脑”。此后不久，他联系了 Phoronix，我了解到了更多他们的计划，并随后评测了 Razer Blade Stealth 笔记本电脑在 Linux 上的表现。但自那以后，Razer 关于 Linux 计划的消息便石沉大海。到 2019 年，他们的 Linux 雄心似乎已被遗忘。

快进到上个月，Razer 主动联系并分享说，他们正在认证新款 Blade 18 笔记本电脑上的 Ubuntu，作为 Microsoft Windows 11 的替代方案。搭载 Intel Core Ultra 9 290HX 和 NVIDIA GeForce RTX 5090 的 Razer Blade 18 提供了强大的计算能力，不仅适合游戏玩家，也适合开发者——无论你是进行代码开发和大量编译，还是利用本地 AI 推理来发挥 GeForce RTX 5090 的性能。

截至撰写本文时，Razer 仍在进行 Blade 18 笔记本电脑在 Ubuntu Linux 上的认证流程。他们向 Phoronix 补充道：“未来支持该平台对我们很重要。”在 Min-Liang Tan 首次公开谈论 Linux 近十年后，我们终于看到第一款 Razer 笔记本电脑正在通过 Ubuntu 的 Linux 认证。

尽管他们正在认证其笔记本电脑在 Ubuntu Linux 上的使用，但现阶段他们尚未宣布任何关于将 Razer Synapse 软件引入 Linux 以更好地管理 Razer 笔记本电脑和其他游戏外设的计划。不过，他们确实承认了社区维护的驱动程序，并指出结合这些驱动（例如 OpenRazer 和 Polychromatic）可以在 Linux 上获得不错的、尽管是非官方的 Razer 体验。

有些令人惊讶的是，当使用来自 Ubuntu PPA 的最新 OpenRazer 和 Polychromatic 时，这款笔记本电脑尚未得到开源社区维护软件的支持。不过话说回来，这些软件主要用于 RGB 灯光控制之类——如果你对这些控制感兴趣的话。

我一直在 Ubuntu 26.04 LTS 上试用 Razer Blade 18 RZ09-0582，运行非常流畅。由于 Linux 上成熟的 Intel Arrow Lake CPU 支持，以及使用官方 NVIDIA Linux 驱动时 GeForce RTX 5090 Blackwell 显卡的良好状态，整个过程一帆风顺。ACPI 平台配置文件和其他功能在这款笔记本电脑上的 Ubuntu 26.04 LTS（搭载 Linux 7.0 内核）中也能正常工作。

Core Ultra 9 290HX Plus 总共拥有 24 个核心，由 8 个 P 核和 16 个 E 核组成。这款 Arrow Lake 处理器具有 5.5GHz 的睿频频率、40MB 智能缓存、55 瓦的基础功耗和 160 瓦的睿频功耗……是的，它性能强劲，但也相当耗电。在 Razer Blade 18 上，最好在接通交流电源时发挥最佳性能，例如主要放在办公桌上使用，通勤时则仅依靠电池。

Razer 提供的 Razer Blade 18 评测机配备了 2×16GB DDR5-6400 内存、2TB NVMe Samsung MZVL82T0HBL1 NVMe SSD、3840×2400 显示屏，以及配备 24GB GDDR7 显存的 NVIDIA GeForce RTX 5090 独立显卡。

这款评测机配置为 Core Ultra 9 290HX Plus / GeForce RTX 5090 / 2TB SSD / 32GB 内存，零售价为 5399 美元。如果升级到 128GB 内存，价格则跃升至 6999 美元。
