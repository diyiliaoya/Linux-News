Canonical 正式推出“Enterprise Store”，用于离线/气隙 Ubuntu 环境

2026年7月21日，美国东部时间上午11:38

几个月来，“Enterprise Store”（企业商店）一直在一些 Ubuntu 文档和其他元素中被提及，而今天 Canonical 正式宣布了它。Enterprise Store 旨在帮助管理 Ubuntu Linux 部署，尤其适用于那些计算机可能处于气隙环境或具有严格互联网控制的企业组织。

Enterprise Store 旨在使软件分发在离线/气隙或存在其他互联网访问限制的环境中更易于管理和扩展。同样适用于受软件版本控制监管且仅允许使用经过验证软件的行业。

Enterprise Store 是 Ubuntu Pro 订阅的一部分，支持本地软件缓存、高可用性访问、细粒度版本控制以及其他功能，帮助 Snaps、Charms 及其他 Ubuntu 软件在防火墙后部署。

对 Canonical 面向 Ubuntu Pro 客户的 Enterprise Store 感兴趣的用户，可以通过今天的公告了解更多信息。
