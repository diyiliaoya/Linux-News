在合并窗口逐渐收尾并最终于昨天发布 Linux 7.2-rc1 之后，我已经开始在更多硬件上进行测试。今天的测试着眼于 Linux 7.2 在其早期开发阶段与 AMD Ryzen Threadripper 上的稳定版 Linux 7.1 相比表现如何。

上周末，我使用截至6月27日的 Git 状态对 Linux 7.1 稳定版和 Linux 7.2 进行了基准测试。两个内核版本在相同系统上使用了相同的标准 Kconfig 配置。该系统是一台 Threadripper HEDT 桌面/工作站，配备 96 核 AMD Ryzen Threadripper PRO 9995WX、ASUS PRO WS TRX50-SAGE WIFI 主板、4 x 32GB DDR5-6400 内存、2TB Corsair MP700 PRO NVMe 固态硬盘以及 Radeon AI PRO R9700 显卡。

两次基准测试运行之间唯一的区别是更换了 Linux 7.1 和 7.2 Git 内核，以观察新内核的性能影响。Linux 7.2 稳定版将于 8 月发布，并将驱动 Ubuntu 26.10 和 Fedora 45 等系统。
