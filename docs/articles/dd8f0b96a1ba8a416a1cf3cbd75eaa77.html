在白天改变夜灯的色温 - OMG! Ubuntu

Night Light Scheduler 是一款新的 GNOME Shell 扩展，让你可以控制屏幕在傍晚（或白天）的暖色程度。它提供可自定义的每日计划，并允许你选择或多或少偏橙色的色温（这将减少设备屏幕发出的蓝光）。

但你所设定的温度在开启后会一直保持不变。它在开始和结束时逐渐过渡，但中间不会变化。

对大多数人来说，默认行为已经足够。这是一种设置后即可忘记的固定计划。

但如果你希望屏幕色温在一天中有所变化，Night Light Scheduler 能为你提供这种能力。

你可以在傍晚早些时候设置较冷的色温，然后每小时逐渐增加暖度，直到睡前达到更深的色调。

为你的显示器设置最多7个色温值（1500K至6500K），在一天中的特定时间使用（以15分钟为增量），并调整它们之间的过渡时间（0至120分钟）。

较长的过渡时间能在色温值之间实现微妙、平滑的切换（更难被注意到）。较短的过渡时间意味着色温值之间更快的变化（瞬间完成，但过渡可能显得突兀）。

计划配置可以导入和导出为可编辑格式。

Night Light Scheduler 适用于 GNOME 46（Ubuntu 24.04 LTS）至 GNOME 50（Ubuntu 26.04 LTS）。你不需要额外安装任何东西，因为它利用了 GNOME 内置的色彩偏移功能。

通过浏览器从 GNOME Extensions 网站安装，或者为了更方便，从 Ubuntu 软件源安装该应用并在其中搜索。
