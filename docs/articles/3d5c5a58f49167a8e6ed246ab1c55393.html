今年早些时候。这是一款基于 MPV 的播放器，拥有沉浸式界面、基本视频功能（字幕、音轨、播放列表）并支持视频调整（亮度、对比度、缩放等）。

六月到来时新增了一个功能。顾名思义，它会显示你最近观看过的历史记录。在没有打开文件的情况下启动应用，主屏幕现在会有一个图标可直接跳转到该功能。

此版本还修复了打开新文件时应用不会取消暂停的问题，以及在某些桌面任务栏上出现空标题的问题。该应用新增了乌克兰语和泰米尔语翻译。

Shotcut 恢复了外接显示器支持

OpenGL 还是 Vulkan？由你决定

开源视频编辑器

在其最新稳定版本 v26.6.25 中增加了外接显示器支持。该功能通过

播放器 > 外接显示器

菜单访问，允许编辑者在第二块屏幕上显示视频预览。

此版本还引入了对 OpenFX 和 VST2/LV2 插件的实验性支持。目前集成还很基础：没有自定义插件 UI，所有内容都在 CPU 上运行，并且默认没有捆绑兼容的插件。

最后，虽然官方发布说明强调了一项新的 HDR 预览窗口功能，但该功能目前无法在 Linux（或 Windows ARM）上运行。

Plank Reloaded 增加了清空垃圾桶音效

，经典的 Plank 扩展坞的复兴版（且支持 Wayland）在六月发布了两个小版本。

其中较奇特的一个为扩展坞增加了音效。当你清空垃圾桶或将文件拖入其中时，你会听到一个声音（有点像 macOS）。这仅在桌面设置中启用了事件音效时才有效，但这是一个可爱的小点缀。

扩展坞的日历弹出窗口在关闭时不再抛出 GTK 关键警告，并为非英语区域本地化了 AM/PM 标记。在“焦点跟随鼠标”设置下运行时，扩展坞不再抢夺焦点。

Plank Reloaded 现已可从项目网站获取

Bazaar 摆好了摊位（依然流畅）

在 Bazaar 0.8.2 中更轻松地找到自适应应用

Bazaar，这款用于管理 Flatpak 和浏览 Flathub 的流畅 GTK4/libadwaita 前端，在最新版本中获得了一些微小但值得赞赏的 UI 调整。

一个新的移动端搜索过滤器使得在较窄宽度下（对于窗口平铺设置同样有用）找到适配良好的软件更加容易。
