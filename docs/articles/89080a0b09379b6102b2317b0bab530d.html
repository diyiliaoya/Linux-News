Ubuntu 将时间同步工具替换为基于 Rust 的版本

Ubuntu 计划将其默认时间同步客户端从 Chrony 替换为基于 Rust 的 NTP 实现，以提升内存安全性和系统稳定性。Canonical 已成为 Trifecta Tech Foundation 的金牌赞助商，每年提供 4 万欧元资助其内存安全软件项目。新工具将在 Ubuntu 26.10 中提供测试，并计划于 Ubuntu 27.04 中成为默认配置。此举是 Ubuntu 全面拥抱 Rust 语言、逐步“氧化”发行版的一部分。
