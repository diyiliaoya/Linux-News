实验性 Holo Core 构建发布：面向 Valve Steam Frame 的 Arch Linux AArch64 版本

2026年7月17日，美国东部时间上午11:59。

作为 Valve 即将推出的高通驱动 Steam Frame 头显的一部分，Valve 一直在与 Collabora 合作，为其平台打造基于 Arch Linux AArch64 的基础系统。今天发布的正是这一“Holo Core”基础系统的初始源代码和二进制文件，该系统基于 Arch Linux AArch64，将用于 Steam Frame。

Collabora 今天宣布了 Holo Core 的初始二进制文件和源代码，这是 Arch Linux 的一个纯 AArch64 移植版，被用作运行 Steam Frame 的操作系统基础。由于上游 Arch Linux 缺乏官方的 AArch64 支持，这还需要额外的工具链和持续集成（CI）基础设施工作。

此次 Holo Core 发布的目的是让其他人能够探索这一 Arch Linux AArch64 移植版。除了源代码外，还提供了二进制软件包和一个 Docker 容器。

这并不是 Arch Linux 针对 AArch64 的完整重建，正如今天博客文章中所解释的：

“现阶段，我们尚未重建整个世界（即 Arch Linux 仓库中所有可用的软件包）。我们的初始目标是提供 Steam Frame 开发和镜像创建所需的一组软件包。即使仅限于 Arch Linux 软件包的这一子集，包括这些软件包及其运行时和构建依赖项，仍然涉及数千个软件包。”
