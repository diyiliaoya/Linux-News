Steam On Linux 使用率在六月略有回落

发布于 2026 年 7 月 1 日 晚上 8:23 EDT。

Steam on Linux 使用率曾飙升至 5.33%，较二月大幅增长 3.1%。在四月，它…… Valve 刚刚发布了六月的 Steam 调查数据，数据显示 Linux 使用率从近期历史高点再次出现小幅回落。

六月的 Steam 调查数据显示 Linux 使用率为 3.69%，较五月下降 0.3%。或者较三月的历史高点下降 1.64%。Linux 使用率下降，而 macOS 上升 0.05% 至 2.21%，Windows 上升 0.25% 至 94.1%。

值得一提的是，在六月的调查中，简体中文用户占比大幅增长 2.34%，达到所有 Steam 用户的 24.19%。通常，当中国 Steam 用户基数大幅增长时，Linux 使用率会相应下降。

SteamOS 仍占所有 Linux 游戏玩家的约 23%。

Linux 游戏玩家中 AMD CPU 使用率略低于 67%。Steam Deck 和 Steam Machine 采用 AMD 定制 SoC，而 AMD CPU 在 Linux 游戏玩家/爱好者中普遍受欢迎，部分原因在于该公司在开源方面的声誉。
