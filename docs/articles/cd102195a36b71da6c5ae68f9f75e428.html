AMD EPYC Turin 搭配 PCIe 5.0 存储 在 Linux 7.2 上展现出显著性能提升

发布于 2026 年 7 月 22 日 中午 12:00（美国东部时间）

本周早些时候，我们看到了一个令人兴奋的测试：Intel Xe3 图形性能在 Linux 7.2 上借助 Intel Core Ultra Series 3 "Panther Lake" 硬件所取得的提升。而在另一边，对于 AMD 硬件而言，在这个即将到来的内核中，一个值得关注的领域是第五代 EPYC 在搭配高速 PCIe Gen5 NVMe SSD 存储时的 I/O 改进。

我继续在更多系统上尝试开发内核，总体而言取得了不错的成功。Linux 7.2 正在稳步推进，预计于 8 月下旬发布稳定版，并带来了众多新特性和改动。Linux 7.2 之所以更加重要，是因为它将成为 Ubuntu 26.10 等发行版的默认内核。

我最新分享的 Linux 7.2 测试，是在一台 AMD EPYC 9455P 48 核服务器上，使用相同的 Kconfig 配置，对 Linux 7.1 稳定版与最新的 Linux 7.2 Git 快照进行基准测试的结果。

这台 AMD EPYC 9455P 服务器基于相应平台构建，存储方面采用了一块三星 PM1743 MZWLO3T8HCLS-00A07 PCIe 5.0 NVMe 固态硬盘。该 EPYC 9455P 服务器配备了 12 条 64GB DDR5-6000 内存，并以 Ubuntu 26.04 为基础操作系统，运行上述用于对比的 Linux 7.1/7.2 内核。
