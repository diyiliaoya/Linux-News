Miracle-wm 桌面新增焦点模糊与夜间护眼模式

基于 Wayland 的平铺窗口管理器 Miracle-wm 发布新版本，新增焦点模糊效果（对未聚焦窗口应用高斯模糊）和夜间护眼模式（根据时间自动调节屏幕色温）。此外，新版本实现了更多 Wayland 协议扩展，原生支持截图工具和屏幕共享，并修复了多个问题。开发者计划年底发布 1.0 正式版。

焦点模糊、夜间护眼模式加入 Miracle-wm 桌面 - OMG! Ubuntu

基于 Wayland 的合成器 Miracle-wm 发布了新版本，它内置了 i3/Sway 风格的平铺窗口管理器。

开发者 Matthew Kosarek 是 Canonical 的工程师，他在业余时间开发这个以键盘驱动的用户体验。他表示，2026 年 4 月版本中引入的插件系统“每天都在变得更好”。

现在，插件可以使用两次分离式高斯模糊着色器，为单个未聚焦窗口设置模糊效果。这有助于提供视觉提示，让你更专注于当前窗口。

新版本利用输出着色器能力，将屏幕色调调整为橙色，减少蓝光发射。该功能按计划运行，从下午 6 点到早上 8 点生效。

此外，还实现了额外的 Wayland 协议扩展。这些扩展为 Miracle-wm 提供了对截图工具等流行实用程序的开箱即用支持，而 xdg-desktop-portal-wlr 则打开了屏幕共享管道。

Miracle-wm 的最新变更集展示

Miracle-wm 0.10.0 中的其他变更和修复：

- GTK4 工具，用于告知配置是否存在错误

- miraclemsg 调试覆盖层，显示窗口和输入区域信息

- 插件自动从 ~/.config/miracle-wm/plugins 加载

- WasmEdge 的 LLVM 不再与导致黑屏的问题冲突

- 现在，即使未在配置中列出，显示器也可正常使用

Miracle-wm 计划在今年年底发布其至关重要的 1.0 版本。

在 Ubuntu 上安装 Miracle-wm：

sudo snap install miracle-wm --classic

适用于 Ubuntu 25.10 或 26.04 LTS——PPA 不再支持 Ubuntu 24.04 LTS。

要使用它，请退出当前会话，然后从 Ubuntu 登录屏幕右下角的齿轮中选择“Miracle”会话。

要退出 Miracle-wm，可以按相应快捷键。

然而，Miracle-wm 并非传统的桌面环境，而仅仅是一个合成器。你需要手动编写文本配置文件来定义偏好，并安装额外的包来获取面板、启动器和小程序。
