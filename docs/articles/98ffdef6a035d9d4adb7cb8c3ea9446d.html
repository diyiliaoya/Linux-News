AWS工程师报告Linux 7.0导致PostgreSQL性能减半，但修复可能不易

一位亚马逊/AWS工程师在周五发出警报，称当前的Linux 7.0开发内核导致PostgreSQL数据库服务器的吞吐量约为先前内核版本的一半。导致PostgreSQL性能减半的罪魁祸首已知，但回滚似乎不太可能发生，目前建议PostgreSQL可能需要进行适配。

Steam on Linux使用量在三月飙升——超过macOS游戏市场份额的两倍

如果Valve最新的Steam调查月度数据准确，那么Steam on Linux在三月经历了一个非常火爆的月份。Steam on Linux现已超过5%的门槛，并且是Steam on macOS市场份额的两倍以上。

Arch Linux现认为恶意软件事件已受控：受影响包超过1500个

当天开始时，Arch Linux的AUR用户贡献仓库发现有超过400个包被恶意软件入侵。到当天结束时，他们认为所有受影响的提交都已被处理。但最终受影响包的数量超过了1500个。

Linux历经六年工作、360多个补丁后终于移除strncpy API

Linux 7.2终于从Linux内核中移除了strncpy API。用于复制指定字节数的strncpy()函数早已被弃用，经过六年的工作和数百个补丁，Linux内核中不再有strncpy接口的用户，因此它现在已被移除。

ReactOS“开源Windows”达到能够运行《半条命》的里程碑

ReactOS，这个致力于与微软Windows计算机程序和驱动程序二进制兼容的开源操作系统，达到了一个里程碑：能够在这个开源平台上运行经典游戏《半条命》。

macOS 27 Beta破坏启动Asahi Linux的能力

Asahi Linux警告其用户不要尝试苹果本周发布的新macOS 27“Golden Gate”测试版。在macOS 27测试版中，Asahi Linux分区不再可见，因此无法启动您的Apple Silicon Linux安装。
