GNOME 51 Alpha 发布，带来众多增强功能

2026年7月3日，美国东部时间下午6:43。

为迎接计划于9月发布的稳定版GNOME 51桌面环境，今天标志着GNOME 51 Alpha版本的发布。

GNOME 51在其众多不同组件中带来了许多变化。在这个阶段，没有哪一项变化特别引人注目，但各种较小的改进为GNOME 50提供了更多的打磨和新增功能。以下是我在GNOME 51 Alpha中注意到的一些变化：

- Epiphany浏览器新增了复制当前页面URL的键盘快捷键，以及其他改进。

- GDM显示管理器现在兼容KMSCON VT。此外，现在还有一个GDM设置，用于更改回退会话。

- GNOME Calendar经过了重新设计，带来了性能改进、渲染优化以及其他内部代码重构。

- GNOME Control Center重新排列了系统软件/硬件信息页面，新增了显示选项，能够在插入鼠标时禁用触摸板，支持用于WiFi共享的GNOME二维码小部件，以及其他新增功能。

- GNOME Maps现在支持离线地图下载。

- GNOME Remote Desktop已修复并启用了AMD显卡驱动程序的硬件加速支持。

- GNOME Session现在支持使用oo7-portal作为Secret门户的实现。

- GNOME Shell现在支持vA-API H.264屏幕录制的速率控制。GNOME Shell现在还支持text_input_v3协议的第2版。

- GTK现在在各个地方尊重减少动效设置，并避免动画。

- Loupe图像查看器现在可以显示图像元数据中的创建者、版权信息、相机镜头和使用的软件。

- 对Mutter合成器进行了许多修复。

- Nautilus文件管理器现在具有增强的视图重载性能，以及其他变化。

- Sushi已移植到GTK4。

- XDG Desktop Portal GNOME现在支持输入捕获门户中的会话持久性。
