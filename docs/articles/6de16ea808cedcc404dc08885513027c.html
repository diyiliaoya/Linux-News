Realtek RTL8723BS WiFi Linux 驱动程序针对恶意 WiFi 接入点进行加固

发布于 2026 年 7 月 12 日，美国东部时间上午 6:15。

本周在 Linux 7.2-rc3 发布前提交的暂存驱动程序修复，主要对 Realtek RTL8723BS WiFi 驱动程序进行了加固。特别是针对连接“不良”WiFi 主机时出现的越界行为进行了多项修复。

Alexandru Hossu 一直在解决一系列越界（OOB）内存访问问题，这些问题可能发生在驱动程序迭代来自空中帧的攻击者控制数据时。该驱动程序未能正确验证头部或有效载荷，如果连接到恶意 WiFi 接入点，可能导致多次 OOB 读取/写入。

这些补丁将在今天合并，赶在 Linux 7.2-rc3 内核发布之前。这些安全修复也将回溯到现有的 Linux 稳定内核版本。这些围绕空中 WiFi 数据处理的的安全缺陷自 2017 年 rtl8723bs 驱动程序被添加到 Linux 内核暂存区以来就一直存在。

自近十年前进入暂存区以来，Realtek RTL8723BS 驱动程序几乎在每个内核周期都持续进行大量清理工作，其代码复杂性堪称“驱动程序中的猛兽”。Realtek RTL8723BS 是一款支持 802.11 b/g/n 的 SDIO WiFi 芯片组，并支持蓝牙 4.0，多年来已被用于许多不同设备。

这些针对恶意 WiFi 主机的越界修复是本周暂存驱动程序修复的主要变更内容。
