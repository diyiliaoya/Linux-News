COSMIC 桌面更新新增系统监视器应用

System76 的 COSMIC 桌面发布 Epoch 1.1.0 更新，主要新增了原生系统监视工具 COSMIC Monitor，该工具基于 Iced 工具包构建，将取代 GNOME 系统监视器。此外，更新还修复了长时间运行导致的 CPU 占用上升问题，支持递归扫描自定义壁纸目录、通过配置文件命名固定工作区，并实现了 pointer-warp-v1 Wayland 协议。其他改进包括排序图标优化、键盘导航增强、MIME 类型图标扩充等。Pop!_OS 24.04 LTS 用户可通过常规更新获取。

COSMIC 桌面更新新增系统监视器应用 - OMG! Ubuntu

System76 的 COSMIC 桌面现已推出新更新，包含一款新的系统监视工具以及一系列修复和微调。

COSMIC Epoch 1.1.0 还标志着这款基于 Rust 的桌面背后的开发者决定“定期递增次要版本号，以便在必要时为中间版本提供补丁版本”。

最大的新功能是 COSMIC Monitor，这是一款原生系统监视工具，使用与其他桌面核心应用相同的 Iced 工具包和小部件集构建。它将取代基于 GTK 的 GNOME 系统监视器，但用户仍可继续安装和使用后者。

COSMIC Monitor（前景），GNOME 系统监视器（背景）

每个版本都会带来一系列针对桌面、底层组件和核心应用的错误修复和功能改进。在 COSMIC 1.1.0 中，值得注意的变化包括“防止长时间运行会话中逐渐变慢和 CPU 使用率上升”。

排序图标表示活动模式

现已支持递归扫描自定义壁纸目录，你可以通过配置文件命名固定工作区，并且系统实现了 pointer-warp-v1 Wayland 协议。

排序操作的图标（之前使用 unicode，但如果桌面字体发生变化，图标也会改变）已启用，键盘导航功能得到增强，MIME 类型应用处理得到改进，以便文件在兼容应用中打开。

更多 MIME 类型图标被添加到标准 COSMIC 图标集中，以帮助用户更好地区分文件。新增了 PDF、ePub、可执行文件以及各种流行的编码、开发和编程文档的图标，包括其他类型。

其他变化包括修复了输入和输出音量匹配时 OSD 不显示的问题，处理了 openvpn 导入中的非文件 URL，以及通过 XDG Portals 实现了“屏幕录制对话框中更好的输出排列”。

Epoch 1.1.0 中（众多）新 MIME 类型图标的一部分

COSMIC Epoch 1.1.0 现已可用。如果你使用的是 Pop!_OS 24.04 LTS，你将通过常规软件更新收到所有更改（事实上，你可能已经收到了——Epoch 只是将它们整合在一起供打包者使用）。

适用于 Ubuntu 26.04 LTS 的非官方 COSMIC PPA 应该会很快更新。
