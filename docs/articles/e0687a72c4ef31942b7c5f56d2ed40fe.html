Linux Mint 23 将提供完整的 Wayland 支持（但 X11 仍会保留）

Linux Mint 宣布其下一个版本（Linux Mint 23）中的 Wayland 支持将不再是实验性质，而是作为完全支持的选项提供。不过，与其他发行版不同，Mint 仍将继续提供并支持 X11。Cinnamon 桌面环境在 Wayland 上进行了多项改进，包括窗口大小和位置映射、防焦点窃取、多显示器支持、HiDPI 渲染优化以及硬件加速等。Linux Mint 23 预计于 2026 年底发布，届时将调整发布节奏，每年减少发布次数以给予开发者更多时间开发新功能。

Linux Mint 表示，其下一个版本中的 Wayland 支持将不再被视为实验性质，而是作为完全支持的选项提供。

然而，与其他已从默认安装中移除传统 Xorg/X11 显示服务器的 Linux 发行版不同——例如 Ubuntu 已放弃 Xorg 支持——Mint 将继续提供并支持 X11。

“我们在 Wayland 上付出了巨大努力，现在已经达到了一个稳定且体验几乎与 X11 相当的水平。”

“X11 和 Wayland 都将得到完全支持。”

至于 Wayland 是否会成为 Linux Mint 23 的默认会话，目前尚不清楚。正如我之前报道的，Lefebvre 表示他希望将这个选项“摆上台面”，但并未承诺切换的时间表。

Cinnamon 的 Wayland 支持得到改进

实验性标签将被移除

Lefebvre 列举了近期对 Cinnamon Wayland 体验所做的一系列改进，他表示这些改进已使其在日常使用中几乎与 X11 持平。

其中包括：Cinnamon on Wayland 现在可以正确映射窗口、小程序和内容菜单的大小与位置；新增了防焦点窃取功能，确保在没有明确用户交互的情况下，输入或点击不会落入其他应用程序。

多显示器设置和硬件 KVM 切换器得到了更好的支持，同时改进的 HiDPI 支持带来了更清晰的文本和图标渲染。硬件加速也已集成到合成器、桌面以及 Wayland 和 XWayland 客户端中。

Root 应用现在可以在 Wayland 上运行（而非通过 XWayland），并且对窗口进度的支持意味着进度指示器（例如复制操作期间任务栏按钮上的进度条）能够正常工作。

除了 Wayland 之外，下一版本的 Cinnamon 还引入了一个新的 cinnamon-list-windows 命令，用于列出所有打开的窗口及相关数据（如位置、大小、HiDPI 支持等），并支持 systemd 的 graphical-session，以提高与其他发行版的兼容性。

Linux Mint 23 将是下一个主要版本，预计于 2026 年底发布，此前 Linux Mint 的发布流程将发生重大变化。该发行版将每年减少发布次数，以便让开发者有更多时间开发新功能。
