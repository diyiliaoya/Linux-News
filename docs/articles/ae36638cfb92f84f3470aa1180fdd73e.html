发行版发布：ArchBang Linux 010726

ArchBang 项目发布了新版本 010726，该版本用 Mango 窗口管理器取代了 labwc Wayland 合成器，并完成了从 FruitBang 到 ArchBang 的品牌重塑。新 ISO 镜像包含最新软件包和 Linux 内核，提供图形化与命令行两种安装方式，其中 arch-browser-installer 仍处于测试阶段，建议仅在虚拟机中测试。

发行版发布：ArchBang Linux 010726 (DistroWatch.com 新闻)

ArchBANG 项目发布了一个新版本的 Linux，并带来了另一项重大变化——它用 Mango 窗口管理器取代了 labwc Wayland 合成器。这个标记为 010726 的新版本同时提供了图形化与命令行系统安装程序：

“很高兴宣布运行 Mango 窗口管理器的 ArchBang 版本发布。这标志着从 FruitBang 到 ArchBang 的全面品牌重塑。一张新的 ISO 镜像已于 7 月 1 日按承诺上传，携带最新的软件包和最新的 Linux 内核。入门指南——点击 waybar 工具栏上的黄色火箭图标即可开始。你所需的大部分内容都可以从该菜单中访问。在 ~/Scripts 目录下，你可以找到一些有用的工具，包括 yay 安装程序和 abi-install（一个 GUI 安装程序，对于终端用户来说是可选的/不需要的，但它会保留）。关于安装程序的说明：arch-browser-installer 仍然是一个进行中的工作，请将其视为测试版，目前仅在虚拟机中测试。abi-install 不会消失；它面向那些对终端不太自信的用户，我们有兴趣继续发展这个想法。下载 ISO 镜像并试用一下。一如既往地欢迎反馈和错误报告。”

ArchBang Linux 是一款基于 Arch Linux 的轻量级发行版。它使用 labwc Wayland 合成器，运行快速、保持最新，适用于桌面和便携系统。除了 GreenBANG，该项目还开发了一个名为 SwayBang 的发行版，采用 Sway Wayland 合成器。
