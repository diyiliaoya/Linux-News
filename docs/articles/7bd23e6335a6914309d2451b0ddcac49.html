发行版发布：Galactic Mandate Linux 99

科幻作家M. R. Richardson宣布发布Galactic Mandate Linux 99。该发行版基于Ubuntu 26.04，搭载Budgie桌面环境和labwc Wayland合成器，新增深度视觉转换、自定义FUI桌面、HUD面板、实时系统遥测、八种书封主题、控制中心等功能，并预装Firefox和OnlyOffice等实用软件。

发行版发布：Galactic Mandate Linux 99 (DistroWatch.com 新闻)

科幻作家、Galactic Mandate系列科幻小说作者M. R. Richardson宣布发布Galactic Mandate Linux 99。该版本基于Ubuntu 26.04，搭载Budgie桌面环境和labwc Wayland合成器，并包含书封主题、实时系统遥测、主题化应用程序、警报模式、可视化文件工具以及实用的Linux软件：

"Galactic Mandate Linux 99版本相比早期预览版增加了更深入的视觉转换。系统现在启动进入一个自定义FUI桌面，带有HUD轨道、实时CPU、内存和交换分区读数、与主题关联的书封壁纸、自定义应用启动器，以及一个首次启动导览，解释各个功能的位置。新的控制中心充当该发行版的指挥室。它允许用户在手工制作的Galactic Mandate主题之间切换，预览桌面语言，控制FUI外壳，以及排练或禁用黄色/红色警报效果。主要特性：基于Ubuntu Budgie、Wayland和labwc构建的太空FUI桌面；八个可选的书封主题，配有协调的GTK、图标、壁纸、外壳镀铬、窗口边框、终端颜色和自定义应用调色板；带有实时遥测、时钟、主题标识、书籍标识和购书控制的HUD外壳……"

关于Galactic Mandate

Galactic Mandate Linux是一款基于Ubuntu的Linux发行版，采用Budgie桌面并带有科幻主题。它包含书封主题、实时系统遥测、主题化应用程序、警报模式、可视化文件工具以及实用的Linux软件，例如以Flatpak包形式提供的Firefox和OnlyOffice。该发行版由科幻作家、Galactic Mandate系列小说作者M. R. Richardson开发。
