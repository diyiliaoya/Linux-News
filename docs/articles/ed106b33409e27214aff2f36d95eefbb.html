Linux 7.2-rc3 发布：接近“新常态”

2026年7月12日，美国东部时间下午5:59。

Linux 7.2-rc3 现已可供测试，以推进将于八月发布的稳定内核版本。

Linux 7.2-rc3 本周的变更包括：

为 RISC-V 默认内核配置添加 Ultra RISC-V 支持；

SEGA Dreamcast 驱动修复；

显示检测改进，以帮助某些多 GPU 系统；

标记 Nick Desaulniers 重返 LLVM Linux 开发。

此外还有各种错误和回归修复的广泛组合——包括各种 AI 辅助补丁。

Linus Torvalds 写道：

“情况继续看起来正常（‘新常态’，提交率略高，尽管我确实感觉到我们正看到这一点被人们开始放暑假所略微平衡）。

大约一半的变更涉及驱动程序——分布在各处。是的，一如既往有 GPU 和网络，但也有相当多的其他杂项驱动噪声。

其余部分同样分布广泛：网络和文件系统是最大的领域，但我们也有一些文档修复（好吧，其中一些只是纯粹的重缩进），一些核心内核，一些架构修复，以及各种工具修复。

没有什么看起来特别可怕或奇怪。”
