Lemonade 11.5 本地 AI 服务器发布，Lemonade Router 功能完成

2026 年 7 月 22 日，美国东部时间下午 6:35。

在上一版本发布仅一周后，Lemonade 11.5 本地 AI 服务器于今日在 AMD 的 Advancing AI 活动中正式发布。该项目由 AMD 支持，采用开源模式。

Lemonade 本地 AI 服务器可轻松在 Windows 和 Linux 系统上，将 LLM 工作负载部署到 Ryzen AI NPU、Radeon GPU 和 CPU 上。Lemonade 11.5 最重要的变化是完成了 Lemonade Router，该路由器可根据定义的策略自动将查询路由到相关模型——甚至支持 LLM 作为路由器，即使用一个小型 LLM 来决定将特定请求路由到哪个模型。

Lemonade Router 允许基于规则、分类器、语义相似性或 LLM 作为路由器策略来引导请求。策略可在 JSON 文件中定义，也可以通过 Lemonade GUI 创建。有关这些路由器功能的更多详细信息，请参见相关文档。

Lemonade 11.5 还引入了服务端作业引擎，允许客户端提交多步骤任务，并通过 /jobs 端点进行管理。通过这些端点，多步骤任务可以暂停、中断、恢复或删除。另一个新特性是，Lemonade 守护进程现在可以作为 MCP 客户端主机，连接外部的 stdio MCP 服务器。
