Firefox 将从九月起将更新频率翻倍

Mozilla 宣布从九月起调整 Firefox 的发布周期，稳定版更新将从每月一次改为每两周一次。此举旨在让发布计划更可预测，并让已就绪的功能更快推送。Mozilla 将此视为一项实验，将监测影响并适时调整。Firefox 154 将是当前周期下的最后一个版本，随后 Firefox 155 于9月1日发布，之后每两周更新一次。尽管节奏加快，未就绪的功能仍会推迟。更快的更新可能帮助 Firefox 与已采用双周更新的 Chromium 系浏览器竞争。现有用户无需任何操作，更新将通过现有渠道自动推送。对于不喜欢频繁更新的用户，Firefox ESR 版本将继续保持较慢的发布节奏，同时定期接收安全更新。Mozilla 表示不会永久维持双周更新，将根据实验结果灵活调整。
