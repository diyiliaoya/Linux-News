AppManager 在 Ubuntu 上无需 FUSE 即可安装 AppImage

AppManager 是一款基于 GTK4/Libadwaita 的新应用，解决了 Ubuntu 上部分 AppImage 需要 FUSE 2 的问题。它通过替代方式挂载 AppImage，支持 SquashFS 和 DwarFS 格式，并提供类似 macOS 的安装流程。用户可拖拽图标安装、导入现有 AppImage、管理启动选项、图标和路径，支持更新检查、命令行参数设置及全局配置。应用本身以 AppImage 形式提供，开源且兼容多数 Linux 发行版。

Ubuntu 26.10 在运行22年后将 dbus-daemon 退役为默认

Ubuntu 26.10 将自2004年以来首次更换其 D-Bus 实现，用 dbus-broker 取代 dbus-daemon。这一变化对终端用户几乎无影响，因为 dbus-broker 是即插即用的替代品，使用相同的协议、配置文件和总线策略。新实现采用异步事件驱动设计，能避免消息丢失并加快查找速度。此前因 Ubuntu 政策不允许重复包共存以及 AppArmor 兼容性问题，切换延迟至今。用户可在 Ubuntu 26.04 LTS 上手动安装测试。

FreeBSD 16 从其基础系统中移除最后一部分 GPL 代码

FreeBSD 16 的开发取得了重要进展：基础系统中最后一部分 GNU GPL 许可代码已被移除。此前，FreeBSD 安装程序已改用其他实现，而 dialog 工具作为最后一个 GPL 组件，其移除请求于今年二月提出，现已合并到 FreeBSD 16.0 的源代码树中。随着 dialog 的移除，FreeBSD 基础系统中不再包含任何 GNU 代码，GNU 子树也被彻底清除。FreeBSD 16.0 预计于 2027 年 12 月发布。
