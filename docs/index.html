BASIC09 编程语言前端为 LLVM 开发完成

46 岁的 BASIC09 编程语言迎来了新的编译器支持，其前端已为 LLVM 编译器栈开发完成。BASIC09 最初于 1980 年为运行 OS-9 操作系统的 Motorola 6809 CPU 设计。借助这个 LLVM 编译器前端，开发者可以为现代软件和硬件编写 BASIC09 代码。开源开发者 Boisy Gene Pitre 宣布了该前端，并探讨将其上游合并到 LLVM 代码库的可能性。该编译器能够进行词法分析、语法分析、构建 AST、执行语义检查，并生成 LLVM IR。不过，LLVM 开发者表示，这种 80 年代的 niche 语言不符合 LLVM 添加新语言的标准。
