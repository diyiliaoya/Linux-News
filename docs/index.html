Ubuntu 将 Livepatch 引入 arm64，实现免重启内核更新

Canonical 首次将 Livepatch 功能引入 Arm64 设备，使运行在 Arm 硬件上的 Ubuntu 系统能够在不重启的情况下应用关键内核安全补丁。Livepatch 是 Ubuntu 隐藏的安全特性之一，需通过 Ubuntu Pro 启用。该功能此前在 arm64 架构上因堆栈跟踪支持不足和工具链缺失而难以实现，经过 Canonical、内核维护者、硬件厂商和超大规模云服务商多年的协作才得以完成。目前支持的 arm64 内核变体包括 Arm 服务器、云和始终在线硬件。但需注意，Livepatch 仅更新内核，常规软件包仍需通过标准流程更新，且 Canonical 建议定期重启。

KaOS 2026.06 发布

KaOS 项目宣布推出 KaOS 2026.06，标志着发行版全面转向 Dinit 初始化系统。经过近三年准备和一年测试，首个完全基于 Dinit 的稳定 ISO 镜像发布。KaOS 现使用 Dinit、turnstile、seatd 进行系统启动、席位和服务管理，但仍保留精简版 systemd 用于 udev 和 tmpfiles。显示管理器改用 greetd 与 tuigreet，取代 sddm。ISO 镜像默认搭载 Niri/Noctalia 桌面，而非 Plasma。用户仍可从仓库安装 Plasma 桌面。
